| `03-chemische-reaktionen-cheatsheet.md` | Gold Badge im Lecture-Quiz |

**Cheat-Sheet Inhalte:**

- **Reaktionsgleichungen:** Edukte → Produkte, Aggregatzustände (s), (l), (g), (aq)
- **Massenerhaltung:** Koeffizienten ausgleichen, Atome zählen
- **Chemisches Gleichgewicht:** Doppelpfeil $\rightleftharpoons$, dynamisch aber konstant
- **Elektrolyte-Tabelle:** Stark (NaCl) / Schwach (CH₃COOH) / Nicht (Zucker)
- **Löslichkeitsregeln:**
  - Löslich: Nitrate, Alkalimetall-Salze, NH₄⁺-Salze
  - Unlöslich: Carbonate, Phosphate, Sulfide (außer Alkalimetalle)
  - Ausnahmen bei Halogeniden: Ag⁺, Pb²⁺, Hg₂²⁺
- **Starke Säuren (Die Großen 6):** HCl, HBr, HI, HNO₃, HClO₄, H₂SO₄
- **Neutralisation:** H⁺ + OH⁻ → H₂O + Salz
- **Gasbildende Reaktionen:** Carbonat → CO₂, Sulfid → H₂S, NH₄⁺ + OH⁻ → NH₃
- **Oxidationszahlen-Regeln:**
  1. Element = 0
  2. Einatomiges Ion = Ladung
  3. O = -2 (außer Peroxide: -1)
  4. H = +1 (außer Hydride: -1)
  5. Summe = 0 (Molekül) oder Ionenladung
- **Redox:** Oxidation = e⁻-Abgabe, Reduktion = e⁻-Aufnahme
- **Spannungsreihe:** Aktivere Metalle verdrängen weniger aktive
