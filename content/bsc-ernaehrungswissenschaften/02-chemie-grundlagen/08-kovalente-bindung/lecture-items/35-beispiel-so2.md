## Beispiel: Schwefeldioxid (SO₂)

**Schwefeldioxid** ist ein klassisches Beispiel für Mesomerie.

### Elektronenzählung

S(6) + 2×O(6) = **18 Valenzelektronen** (9 Paare)

### Das Problem

Wenn wir die Lewis-Formel zeichnen, gibt es **zwei Möglichkeiten**, wo die Doppelbindung sitzt:

**Grenzstruktur 1:**

```text
O = S — O :
:       :
```

**Grenzstruktur 2:**

```text
: O — S = O
  :       :
```

### Was ist die "wahre" Struktur?

Experimentelle Messungen zeigen:

- Beide S—O Bindungen sind **exakt gleich lang** (143 pm)
- Beide Bindungen sind **gleich stark**

Das bedeutet: Das Molekül ist **weder** Struktur 1 **noch** Struktur 2, sondern ein **Hybrid** aus beiden!

### Korrekte Notation

```text
O = S — O :   ↔   : O — S = O
:       :           :       :
```

### Bindungsordnung

Da die Doppelbindung im Hybrid "verschmiert" ist:

$$\text{Bindungsordnung} = \frac{1 + 2}{2} = 1{,}5$$

Jede S—O Bindung ist "anderthalbfach" – stärker als eine Einfachbindung, aber schwächer als eine Doppelbindung.

> **Merke:** Äquivalente Grenzstrukturen tragen gleich viel zum Hybrid bei. Die realen Bindungen liegen "dazwischen".
