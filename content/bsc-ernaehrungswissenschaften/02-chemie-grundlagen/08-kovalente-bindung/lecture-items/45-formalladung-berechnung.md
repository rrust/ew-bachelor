## Formalladungen berechnen: Beispiele

### Beispiel 1: Wasser (H₂O)

```text
H — O — H
    : :
```

**Berechnung für O:**
- Gruppenzahl: 6
- Bindungen (Striche): 2
- Freie Elektronen (Punkte): 4

$$FL(O) = 6 - 2 - 4 = 0$$

**Berechnung für H:**
$$FL(H) = 1 - 1 - 0 = 0$$

→ Alle Formalladungen sind 0 ✓

### Beispiel 2: Hydronium-Ion (H₃O⁺)

```text
    H
    |
H — O — H
    :
```

**Berechnung für O:**
- Gruppenzahl: 6
- Bindungen: 3
- Freie Elektronen: 2

$$FL(O) = 6 - 3 - 2 = +1$$

**Berechnung für H:**
$$FL(H) = 1 - 1 - 0 = 0$$

**Summe:** 0 + 0 + 0 + 1 = **+1** ✓ (entspricht der Ionenladung)

### Beispiel 3: Ammonium-Ion (NH₄⁺)

**N:** 4 Bindungen, 0 freie Elektronen
$$FL(N) = 5 - 4 - 0 = +1$$

**H:** je 1 Bindung
$$FL(H) = 1 - 1 - 0 = 0$$

**Summe:** 1 + 0 + 0 + 0 + 0 = **+1** ✓

> **Merke:** Die Summe der Formalladungen muss immer der Gesamtladung entsprechen – das ist eine gute Kontrolle!
