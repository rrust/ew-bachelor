## Beispiel: Thiocyanat-Ion (SCN⁻)

Das **Thiocyanat-Ion** ist ein anspruchsvolles Beispiel für die Anwendung von Formalladungen.

### Valenzelektronen

S(6) + C(4) + N(5) + 1 (Ladung) = **16 Elektronen** (8 Paare)

### Mögliche Resonanzstrukturen

Es gibt drei Hauptstrukturen mit unterschiedlichen Formalladungen:

**Struktur 1: S—C≡N (Dreifachbindung zu N)**

| Atom | Bindungen | Freie e⁻ | FL             |
| ---- | --------- | -------- | -------------- |
| S    | 1         | 6        | 6−1−6 = **−1** |
| C    | 4         | 0        | 4−4−0 = **0**  |
| N    | 3         | 2        | 5−3−2 = **0**  |

Summe: −1 ✓

**Struktur 2: S=C=N (zwei Doppelbindungen)**

| Atom | Bindungen | Freie e⁻ | FL             |
| ---- | --------- | -------- | -------------- |
| S    | 2         | 4        | 6−2−4 = **0**  |
| C    | 4         | 0        | 4−4−0 = **0**  |
| N    | 2         | 4        | 5−2−4 = **−1** |

Summe: −1 ✓

**Struktur 3: S≡C—N (Dreifachbindung zu S)**

| Atom | Bindungen | Freie e⁻ | FL             |
| ---- | --------- | -------- | -------------- |
| S    | 3         | 2        | 6−3−2 = **+1** |
| C    | 4         | 0        | 4−4−0 = **0**  |
| N    | 1         | 6        | 5−1−6 = **−2** |

Summe: −1 ✓

### Stabilitätsvergleich

| Struktur    | Formalladungen | Bewertung                         |
| ----------- | -------------- | --------------------------------- |
| 1: S⁻—C≡N   | −1, 0, 0       | Gut (S kann Ladung stabilisieren) |
| 2: S=C=N⁻   | 0, 0, −1       | Gut (N ist elektronegativ)        |
| 3: S⁺≡C—N²⁻ | +1, 0, −2      | Ungünstig (hohe Ladungstrennung)  |

> **Merke:** Strukturen 1 und 2 tragen beide zum Hybrid bei. Struktur 3 ist wegen der hohen Ladungen weniger wichtig.
