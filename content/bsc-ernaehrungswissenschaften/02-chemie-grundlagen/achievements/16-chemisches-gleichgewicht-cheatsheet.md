# Chemisches Gleichgewicht: Cheat-Sheet

Herzlichen Glückwunsch! Du hast das Quiz mit Gold-Status bestanden.

***

## Eigenschaften des Gleichgewichts

| Eigenschaft    | Bedeutung                                     |
| -------------- | --------------------------------------------- |
| **Dynamisch**  | Reaktionen laufen weiter, aber gleich schnell |
| **Reversibel** | Von beiden Seiten erreichbar                  |
| **Konstant**   | Konzentrationen ändern sich nicht mehr        |

***

## Massenwirkungsgesetz (MWG)

$$K_c = \frac{[\text{Produkte}]^{\text{Koeff.}}}{[\text{Edukte}]^{\text{Koeff.}}}$$

**Regeln:**
- Produkte → Zähler
- Edukte → Nenner
- Koeffizienten → Exponenten
- Feststoffe/reine Flüssigkeiten → weglassen!

***

## Reaktionsquotient Q

| Vergleich | Reaktionsrichtung |
| --------- | ----------------- |
| Q < K     | → nach rechts     |
| Q = K     | Gleichgewicht     |
| Q > K     | ← nach links      |

***

## ICE-Tabelle

| Zeile | Bedeutung                                |
| ----- | ---------------------------------------- |
| **I** | Initial (Anfangskonzentrationen)         |
| **C** | Change (Änderung: ±x nach Stöchiometrie) |
| **E** | Equilibrium (= I + C)                    |

***

## Kp vs Kc (Gase)

$$K_p = K_c \cdot (RT)^{\Delta n}$$

$$\Delta n = \text{Mol Gas Produkte} - \text{Mol Gas Edukte}$$

Wenn Δn = 0 → Kp = Kc

***

## Le Chatelier

| Störung               | Reaktion des Systems     |
| --------------------- | ------------------------ |
| Edukt zugeben         | → rechts                 |
| Produkt entfernen     | → rechts                 |
| Druck erhöhen         | → weniger Gasteilchen    |
| T erhöhen (exotherm)  | ← links, K sinkt         |
| T erhöhen (endotherm) | → rechts, K steigt       |
| Katalysator           | Schneller, K unverändert |

***

## K-Manipulation

| Operation           | Neues K |
| ------------------- | ------- |
| Reaktion umkehren   | 1/K     |
| Koeffizienten × n   | K^n     |
| Reaktionen addieren | K₁ · K₂ |

***

## Thermodynamik

$$\Delta G° = -RT \ln K$$

| K   | ΔG° | Bedeutung          |
| --- | --- | ------------------ |
| > 1 | < 0 | Produkte bevorzugt |
| < 1 | > 0 | Edukte bevorzugt   |
| = 1 | = 0 | Gleichgewicht      |
