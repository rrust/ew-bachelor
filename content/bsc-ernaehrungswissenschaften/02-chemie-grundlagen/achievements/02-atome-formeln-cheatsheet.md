# Atome & Stöchiometrie – Cheat-Sheet

## Subatomare Teilchen

| Teilchen | Symbol | Ladung | Masse (u) | Ort   |
| -------- | ------ | ------ | --------- | ----- |
| Proton   | $p^+$  | +1     | 1,007     | Kern  |
| Neutron  | $n^0$  | 0      | 1,009     | Kern  |
| Elektron | $e^-$  | -1     | 0,0005    | Hülle |

## Nuklid-Schreibweise

$${}^{A}_{Z}\text{E}$$

- **A** (oben) = Massenzahl = Protonen + Neutronen
- **Z** (unten) = Ordnungszahl = Protonen
- Neutronen = A - Z

| Gruppe | Name             | Typische Ladung | Beispiele    |
| ------ | ---------------- | --------------- | ------------ |
| 1      | Alkalimetalle    | +1              | Li, Na, K    |
| 2      | Erdalkalimetalle | +2              | Mg, Ca, Ba   |
| 17     | Halogene         | -1              | F, Cl, Br, I |
| 18     | Edelgase         | 0               | He, Ne, Ar   |

## Coulomb-Gesetz

$$F \propto \frac{q_1 \times q_2}{d^2}$$

- **Höhere Ladung** → stärkere Bindung
- **Kleinerer Abstand** → stärkere Bindung

## Das Mol

**Avogadro-Konstante:**
$$N_A = 6,022 \times 10^{23} \text{ mol}^{-1}$$

**Umrechnungen:**

| Von → Nach     | Formel             |
| -------------- | ------------------ |
| Gramm → Mol    | $n = \frac{m}{M}$  |
| Mol → Gramm    | $m = n \times M$   |
| Mol → Teilchen | $N = n \times N_A$ |

## Formelermittlung

### Empirische Formel

1. Basis 100 g annehmen
2. Massen → Mol umrechnen
3. Durch kleinste Molzahl teilen
4. Ganzzahlig machen

### Molekulare Formel

$$\text{Faktor} = \frac{M_{\text{real}}}{M_{\text{empirisch}}}$$

## Nomenklatur ionischer Verbindungen

**Kation zuerst, Anion zuletzt!**

- Variable Ladung (Fe, Cu, etc.): Römische Zahl angeben
  - $FeCl_2$ → Eisen(II)-chlorid
  - $FeCl_3$ → Eisen(III)-chlorid

- Anionen: -id (Chlorid), -at (Sulfat), -it (Sulfit)
