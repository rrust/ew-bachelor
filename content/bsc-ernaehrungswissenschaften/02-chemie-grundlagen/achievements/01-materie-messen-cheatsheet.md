## Wichtige Formeln

### Dichte

$$\rho = \frac{m}{V}$$

- $\rho$ = Dichte (g/cm³)
- $m$ = Masse (g)
- $V$ = Volumen (cm³)

> **Merke:** $1 \, \text{mL} = 1 \, \text{cm}^3$

### Temperatur-Umrechnung

$$T(K) = t(°C) + 273{,}15$$

| Fixpunkt            | Celsius   | Kelvin |
| ------------------- | --------- | ------ |
| Absoluter Nullpunkt | −273,15°C | 0 K    |
| Gefrierpunkt Wasser | 0°C       | 273 K  |
| Siedepunkt Wasser   | 100°C     | 373 K  |

## Signifikante Stellen

### Regeln

1. **Führende Nullen** zählen NIE (0,00357 → 3 Sig Figs)
2. **Nullen dazwischen** zählen IMMER (3,0604 → 5 Sig Figs)
3. **Endnullen** nach Dezimalpunkt zählen (2,340 → 4 Sig Figs)
4. **Exakte Zahlen** haben ∞ Sig Figs (12 Stück, 1000 mm/m)

### Rechenregeln

| Operation | Regel                             |
| --------- | --------------------------------- |
| × und ÷   | Wenigste **signifikante Stellen** |
| + und −   | Wenigste **Nachkommastellen**     |

## Klassifizierung von Materie

| Kategorie     | Unterkategorie   | Beispiele                     |
| ------------- | ---------------- | ----------------------------- |
| **Reinstoff** | Element          | Au, O₂ (nicht trennbar)       |
|               | Verbindung       | H₂O, NaCl (chemisch trennbar) |
| **Mischung**  | Homogen (Lösung) | Luft, Zuckerwasser            |
|               | Heterogen        | Sand + Wasser, Öl + Wasser    |
