# Chemische Reaktionen – Cheat-Sheet

## Reaktionsgleichungen

**Aufbau:** Edukte → Produkte

**Aggregatzustände:**

| Symbol | Bedeutung                  |
| ------ | -------------------------- |
| (s)    | solid – fest               |
| (l)    | liquid – flüssig           |
| (g)    | gas – gasförmig            |
| (aq)   | aqueous – in Wasser gelöst |

**Massenerhaltung:** Atome werden weder erschaffen noch vernichtet – nur umgruppiert!

- **Koeffizienten** (vor der Formel): Mengenverhältnis – dürfen angepasst werden
- **Subskripte** (in der Formel): Identität des Moleküls – niemals ändern!

## Chemisches Gleichgewicht

- **Einfachpfeil** ($\rightarrow$): Reaktion läuft vollständig
- **Doppelpfeil** ($\rightleftharpoons$): Reversible Reaktion, Gleichgewicht

**Dynamisches Gleichgewicht:** Hin- und Rückreaktion laufen gleich schnell → keine Nettoänderung

## Elektrolyte

| Typ         | Dissoziation | Leitfähigkeit | Pfeil                |
| ----------- | ------------ | ------------- | -------------------- |
| **Stark**   | ~100%        | Hoch          | $\rightarrow$        |
| **Schwach** | < 5%         | Gering        | $\rightleftharpoons$ |
| **Nicht**   | 0%           | Keine         | —                    |

**Beispiele:**
- Stark: NaCl, HCl, NaOH
- Schwach: CH₃COOH, NH₃
- Nicht: Zucker, Ethanol

## Löslichkeitsregeln

**IMMER löslich:**
- Nitrate ($NO_3^-$)
- Alkalimetall-Salze (Na⁺, K⁺)
- Ammonium-Salze (NH₄⁺)

**MEIST löslich (mit Ausnahmen):**
- Halogenide → außer Ag⁺, Pb²⁺, Hg₂²⁺
- Sulfate → außer Ba²⁺, Pb²⁺, Ca²⁺, Sr²⁺

**MEIST unlöslich:**
- Carbonate ($CO_3^{2-}$)
- Phosphate ($PO_4^{3-}$)
- Sulfide ($S^{2-}$)
- Hydroxide ($OH^-$) → außer Alkalimetalle

## Die 6 Starken Säuren

| Säure                | Formel |
| -------------------- | ------ |
| Salzsäure            | HCl    |
| Bromwasserstoffsäure | HBr    |
| Iodwasserstoffsäure  | HI     |
| Salpetersäure        | HNO₃   |
| Perchlorsäure        | HClO₄  |
| Schwefelsäure        | H₂SO₄  |

**Neutralisation:** Säure + Base → Salz + Wasser

## Gasbildende Reaktionen

| Edukte           | Gas |
| ---------------- | --- |
| Carbonat + Säure | CO₂ |
| Sulfid + Säure   | H₂S |
| Sulfit + Säure   | SO₂ |
| NH₄⁺ + Base      | NH₃ |

## Oxidationszahlen – 5 Regeln

1. **Element:** Oxidationszahl = 0
2. **Einatomiges Ion:** = Ionenladung
3. **Sauerstoff:** = −2 (außer Peroxide: −1)
4. **Wasserstoff:** = +1 (außer Hydride: −1)
5. **Summe:** = 0 (Molekül) oder Ionenladung

## Redox-Reaktionen

| Prozess       | Elektronen | Oxidationszahl |
| ------------- | ---------- | -------------- |
| **Oxidation** | Abgabe     | steigt ↑       |
| **Reduktion** | Aufnahme   | sinkt ↓        |

**Merke:**
- **Reduktionsmittel** wird oxidiert (gibt e⁻ ab)
- **Oxidationsmittel** wird reduziert (nimmt e⁻ auf)

## Spannungsreihe (Auswahl)

↑ Reaktiv (unedle Metalle)

$$Li > K > Ca > Na > Mg > Al > Zn > Fe > Ni > Sn > Pb$$

$$─────────────── H_2 ───────────────$$

$$Cu > Hg > Ag > Pt > Au$$

↓ Edel (Edelmetalle)

- **Unedle Metalle** (über H₂): Reagieren mit Säuren → H₂
- **Edelmetalle** (unter H₂): Reagieren NICHT mit einfachen Säuren
