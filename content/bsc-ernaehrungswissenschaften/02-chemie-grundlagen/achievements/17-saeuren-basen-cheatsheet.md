# Säuren & Basen: Cheat-Sheet

Herzlichen Glückwunsch! Du hast das Quiz mit Gold-Status bestanden.

***

## Brønsted-Lowry Definition

| Begriff   | Definition                      |
| --------- | ------------------------------- |
| **Säure** | Protonendonator (gibt H⁺ ab)    |
| **Base**  | Protonenakzeptor (nimmt H⁺ auf) |

**Konjugierte Paare:** Unterscheiden sich um genau 1 H⁺

***

## Starke vs. Schwache Säuren

| Starke Säuren           | Schwache Säuren         |
| ----------------------- | ----------------------- |
| HCl, HNO₃, H₂SO₄, HClO₄ | CH₃COOH, HF, HCN, H₂CO₃ |
| 100% Dissoziation       | < 5% Dissoziation       |
| → Reaktionspfeil        | ⇌ Gleichgewichtspfeil   |

**Merkregel:** Je stärker die Säure, desto schwächer die konjugierte Base!

***

## pH-Wert Formeln

| Formel                                     | Bedeutung                  |
| ------------------------------------------ | -------------------------- |
| $\text{pH} = -\log[\text{H}_3\text{O}^+]$  | pH aus Konzentration       |
| $[\text{H}_3\text{O}^+] = 10^{-\text{pH}}$ | Konzentration aus pH       |
| $\text{pH} + \text{pOH} = 14$              | Fundamentaler Zusammenhang |
| $K_w = 10^{-14}$                           | Ionenprodukt des Wassers   |

***

## pH schwacher Säuren

**Näherungsformel:**

$$[\text{H}_3\text{O}^+] \approx \sqrt{K_s \cdot C_0}$$

**Gültig wenn:** $100 \cdot K_s < C_0$

**Zusammenhang Ks/Kb:**

$$K_s \cdot K_b = K_w = 10^{-14}$$
$$pK_s + pK_b = 14$$

***

## Salzhydrolyse

| Salz-Typ                     | Beispiel | pH            |
| ---------------------------- | -------- | ------------- |
| Starke Base + Starke Säure   | NaCl     | 7 (neutral)   |
| Starke Base + Schwache Säure | NaCH₃COO | > 7 (basisch) |
| Schwache Base + Starke Säure | NH₄Cl    | < 7 (sauer)   |

***

## Lewis-Theorie

| Begriff         | Definition              |
| --------------- | ----------------------- |
| **Lewis-Säure** | Elektronenpaar-Akzeptor |
| **Lewis-Base**  | Elektronenpaar-Donator  |

**Beispiel:** BF₃ (Säure) + NH₃ (Base) → F₃B-NH₃
