# Einflussfaktoren auf das Gleichgewicht

Le Chatelier beschreibt drei Wege, ein Gleichgewicht zu beeinflussen.

## 1. Konzentration

| Aktion            | Reaktion des Systems                            |
| ----------------- | ----------------------------------------------- |
| Edukt zugeben     | → Verschiebung nach rechts (Edukt verbrauchen)  |
| Edukt entfernen   | ← Verschiebung nach links (Edukt nachbilden)    |
| Produkt zugeben   | ← Verschiebung nach links (Produkt verbrauchen) |
| Produkt entfernen | → Verschiebung nach rechts (Produkt nachbilden) |

**Beispiel:** Fe³⁺ + SCN⁻ ⇌ Fe(SCN)²⁺ (rot)
- Zugabe von Fe³⁺ → mehr rotes Produkt → intensivere Farbe

## 2. Druck/Volumen (nur bei Gasen!)

| Aktion        | Reaktion des Systems                             |
| ------------- | ------------------------------------------------ |
| Druck erhöhen | → Verschiebung zur Seite mit weniger Gasteilchen |
| Druck senken  | → Verschiebung zur Seite mit mehr Gasteilchen    |

**Beispiel:** N₂O₄(g) ⇌ 2NO₂(g)
- Links: 1 Teilchen, Rechts: 2 Teilchen
- Druckerhöhung → Verschiebung nach links (N₂O₄)

**Wichtig:** Wenn Δn = 0, hat Druck keinen Einfluss!

## 3. Temperatur

Hier müssen wir wissen, ob die Reaktion exotherm oder endotherm ist.

**Exotherme Reaktion** (ΔH < 0): Wärme ist ein "Produkt"
$$\text{A} \rightleftharpoons \text{B} + \text{Wärme}$$

| Aktion   | Reaktion                |
| -------- | ----------------------- |
| Erwärmen | ← nach links, K sinkt   |
| Abkühlen | → nach rechts, K steigt |

**Endotherme Reaktion** (ΔH > 0): Wärme ist ein "Edukt"
$$\text{A} + \text{Wärme} \rightleftharpoons \text{B}$$

| Aktion   | Reaktion                |
| -------- | ----------------------- |
| Erwärmen | → nach rechts, K steigt |
| Abkühlen | ← nach links, K sinkt   |

> **Merksatz:** Das System "flieht" vor der Wärme bei exothermen und "sucht" die Wärme bei endothermen Reaktionen.
