# Das Haber-Bosch-Verfahren

Das Haber-Bosch-Verfahren zur Ammoniaksynthese ist eine der wichtigsten industriellen Anwendungen des chemischen Gleichgewichts.

## Die Reaktion

$$\text{N}_2\text{(g)} + 3\text{H}_2\text{(g)} \rightleftharpoons 2\text{NH}_3\text{(g)} + \text{Wärme}$$

$$\Delta H° = -92{,}2 \text{ kJ/mol}$$

## Das Problem

Bei Raumtemperatur ist K sehr groß (10⁵) – thermodynamisch günstig! ABER: Die Reaktion ist extrem langsam, weil die N≡N-Dreifachbindung sehr stabil ist.

## Die Optimierung nach Le Chatelier

### 1. Hoher Druck (150-300 bar)

- Links: 1 + 3 = 4 Gasteilchen
- Rechts: 2 Gasteilchen
- Hoher Druck → Verschiebung nach rechts ✓

### 2. Entfernen von NH₃

- Produkt wird abgeführt (durch Verflüssigung)
- → Verschiebung nach rechts ✓

### 3. Temperatur: Der Kompromiss

| Temperatur | Ausbeute         | Geschwindigkeit |
| ---------- | ---------------- | --------------- |
| Niedrig    | Hoch (exotherm!) | Zu langsam      |
| Hoch       | Niedrig          | Schnell         |
| **~450°C** | **Mittel**       | **Akzeptabel**  |

Die Betriebstemperatur ist ein Kompromiss zwischen Thermodynamik und Kinetik.

### 4. Katalysator (Eisen)

- Beschleunigt die Reaktion erheblich
- Ermöglicht niedrigere Temperaturen
- Ändert K nicht, nur die Geschwindigkeit

## Typische Betriebsbedingungen

| Parameter              | Wert                           |
| ---------------------- | ------------------------------ |
| Temperatur             | 400-500°C                      |
| Druck                  | 150-300 bar                    |
| Katalysator            | Fe mit Promotoren              |
| Ausbeute pro Durchgang | ~15%                           |
| Kreislaufführung       | Ja (unreagiertes N₂/H₂ zurück) |

## Bedeutung

Das Haber-Bosch-Verfahren:
- Produziert ~150 Millionen Tonnen NH₃/Jahr
- Basis für Düngemittel (ernährt ~50% der Weltbevölkerung)
- Nobelpreis 1918 für Fritz Haber
