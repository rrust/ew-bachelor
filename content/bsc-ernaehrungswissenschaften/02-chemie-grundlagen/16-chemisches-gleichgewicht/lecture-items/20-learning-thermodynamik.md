# Thermodynamik und Gleichgewicht

Die Gleichgewichtskonstante K ist direkt mit der freien Standardenthalpie ΔG° verknüpft.

## Die fundamentale Gleichung

$$\Delta G° = -RT \ln K$$

| Symbol | Bedeutung               | Einheit           |
| ------ | ----------------------- | ----------------- |
| ΔG°    | Freie Standardenthalpie | J/mol oder kJ/mol |
| R      | Gaskonstante            | 8,314 J/(mol·K)   |
| T      | Absolute Temperatur     | K (Kelvin)        |
| K      | Gleichgewichtskonstante | dimensionslos     |

## Der Zusammenhang in Zahlen

Bei 25°C (298 K):

| K    | ln K  | ΔG° (kJ/mol) | Interpretation |
| ---- | ----- | ------------ | -------------- |
| 10⁶  | 13,8  | −34          | Stark exergon  |
| 10³  | 6,9   | −17          | Exergon        |
| 1    | 0     | 0            | Gleichgewicht  |
| 10⁻³ | −6,9  | +17          | Endergon       |
| 10⁻⁶ | −13,8 | +34          | Stark endergon |

## Praktische Anwendung

**Beispiel:** Für eine Reaktion mit K = 100 bei 25°C:

$$\Delta G° = -8{,}314 \cdot 298 \cdot \ln(100)$$
$$\Delta G° = -8{,}314 \cdot 298 \cdot 4{,}61$$
$$\Delta G° = -11{,}4 \text{ kJ/mol}$$

Negative ΔG° → Reaktion ist thermodynamisch begünstigt.

## Wichtiger Hinweis

> **K sagt nichts über die Geschwindigkeit!** Eine Reaktion kann ein sehr großes K haben, aber trotzdem extrem langsam sein (z.B. Diamant → Graphit).
