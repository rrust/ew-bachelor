# Das Prinzip von Le Chatelier

Das Prinzip von Le Chatelier (auch "Prinzip des kleinsten Zwanges") beschreibt, wie ein System im Gleichgewicht auf äußere Störungen reagiert.

## Die Grundregel

> **Wenn ein System im Gleichgewicht gestört wird, verschiebt sich das Gleichgewicht so, dass es der Störung entgegenwirkt.**

Das System "wehrt sich" gegen die Änderung!

## Arten von Störungen

Ein Gleichgewicht kann durch drei Faktoren gestört werden:

1. **Konzentration** – Zugabe oder Entfernen von Stoffen
2. **Druck/Volumen** – Bei Gasreaktionen
3. **Temperatur** – Erwärmen oder Abkühlen

## Wichtiger Unterschied

| Störung        | Auswirkung auf K   |
| -------------- | ------------------ |
| Konzentration  | K bleibt gleich    |
| Druck/Volumen  | K bleibt gleich    |
| **Temperatur** | **K ändert sich!** |

## Analogie: Der Körper

Le Chatelier funktioniert wie die Homöostase im Körper:
- Körpertemperatur steigt → Schwitzen beginnt (Gegenwirkung)
- Blutzucker sinkt → Hungergefühl (Gegenwirkung)

Das chemische System verhält sich ähnlich – es versucht, die Störung auszugleichen.

## Anwendung

Le Chatelier hilft vorherzusagen:
- Wie man die Ausbeute eines Produkts steigern kann
- Was passiert, wenn Reaktionsbedingungen geändert werden
- Wie industrielle Prozesse optimiert werden können
