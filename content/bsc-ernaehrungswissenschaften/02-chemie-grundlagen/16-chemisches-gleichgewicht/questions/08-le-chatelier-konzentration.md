type: 'multiple-choice-multiple'
question: 'Für die Reaktion A(g) + B(g) ⇌ C(g) + D(g): Welche Konzentrationsänderungen verschieben das Gleichgewicht nach rechts?'
options:
  - 'Zugabe von A'
  - 'Entfernen von C'
  - 'Zugabe von D'
  - 'Entfernen von B'
correctAnswers:
  - 'Zugabe von A'
  - 'Entfernen von C'
explanation: 'Nach Le Chatelier: Edukt zugeben → rechts. Produkt entfernen → rechts. Produkt zugeben (D) → links. Edukt entfernen (B) → links.'
