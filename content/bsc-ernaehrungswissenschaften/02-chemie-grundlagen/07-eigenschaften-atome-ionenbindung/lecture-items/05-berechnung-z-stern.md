# Berechnung von $Z^*$ – Beispiele

## Vereinfachte Methode

Für eine schnelle Schätzung ziehen wir einfach die Anzahl der inneren Elektronen von der Kernladung ab:

$$Z^* \approx Z - \text{Innere Elektronen}$$

## Beispielrechnungen

### Lithium (Li)

- Ordnungszahl: $Z = 3$
- Konfiguration: $1s^2 \, 2s^1$
- Innere Elektronen: 2 (die 1s-Elektronen)

$$Z^*_{Li} = 3 - 2 = +1$$

### Beryllium (Be)

- Ordnungszahl: $Z = 4$
- Konfiguration: $1s^2 \, 2s^2$
- Innere Elektronen: 2

$$Z^*_{Be} = 4 - 2 = +2$$

### Bor (B)

- Ordnungszahl: $Z = 5$
- Konfiguration: $1s^2 \, 2s^2 \, 2p^1$
- Innere Elektronen: 2

$$Z^*_{B} = 5 - 2 = +3$$

## Präzisere Werte (Slater-Regeln)

Die tatsächlichen Werte weichen etwas ab, zeigen aber den gleichen Trend:

| Element | Geschätzt | Slater-Wert |
| ------- | --------- | ----------- |
| Li      | +1        | +1,28       |
| B       | +3        | +2,58       |
| C       | +4        | +3,22       |
| F       | +7        | +5,13       |

> **Fazit:** Die Valenzelektronen im Fluor spüren eine etwa 5× stärkere Anziehung als im Lithium!
