# Die drei Haupttrends im Überblick

Bevor wir in die Details gehen, hier eine Übersicht der drei Haupttrends:

## 1. Atomradien

| Richtung     | Trend        | Erklärung                   |
| ------------ | ------------ | --------------------------- |
| In Periode → | **nimmt ab** | Höhere effektive Kernladung |
| In Gruppe ↓  | **nimmt zu** | Neue Hauptschalen           |

## 2. Ionisierungsenergie (IE)

| Richtung     | Trend      | Erklärung                        |
| ------------ | ---------- | -------------------------------- |
| In Periode → | **steigt** | Elektronen fester gebunden       |
| In Gruppe ↓  | **sinkt**  | Valenzelektronen weiter vom Kern |

## 3. Elektronenaffinität (EA)

| Richtung     | Trend                    | Erklärung                              |
| ------------ | ------------------------ | -------------------------------------- |
| In Periode → | **wird negativer**       | Stärkere Anziehung für neue Elektronen |
| In Gruppe ↓  | **wird weniger negativ** | Schwächere Anziehung                   |

> **Merke:** Die effektive Kernladung $Z^*$ ist der Schlüssel zum Verständnis aller drei Trends!
