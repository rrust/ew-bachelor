# Mehrfache Ionisierung

## Man kann mehrere Elektronen entfernen

Einem Atom können sukzessive mehrere Elektronen entrissen werden:

$$X \xrightarrow{I_1} X^+ \xrightarrow{I_2} X^{2+} \xrightarrow{I_3} X^{3+} \xrightarrow{I_4} \ldots$$

## Die Ionisierungsenergien steigen

| Ionisierungsenergie | Prozess         | Warum höher?            |
| ------------------- | --------------- | ----------------------- |
| $I_1$ (1. IE)       | Atom → +1 Ion   | Grundwert               |
| $I_2$ (2. IE)       | +1 Ion → +2 Ion | Ion ist bereits positiv |
| $I_3$ (3. IE)       | +2 Ion → +3 Ion | Noch stärkere Ladung    |

## Beispiel: Magnesium

| IE    | Energie (kJ/mol) | Prozess     |
| ----- | ---------------- | ----------- |
| $I_1$ | 738              | Mg → Mg⁺    |
| $I_2$ | 1451             | Mg⁺ → Mg²⁺  |
| $I_3$ | 7733             | Mg²⁺ → Mg³⁺ |

## Warum wird es immer schwerer?

Bei jeder Ionisierung:
1. Das Verhältnis **Protonen : Elektronen** steigt
2. Die verbleibenden Elektronen werden **stärker angezogen**
3. Man braucht **mehr Energie**, um sie zu entfernen

> **Beispiel $I_2$:** Beim Mg⁺ gibt es 12 Protonen, aber nur noch 11 Elektronen. Die Anziehung pro Elektron ist höher!
