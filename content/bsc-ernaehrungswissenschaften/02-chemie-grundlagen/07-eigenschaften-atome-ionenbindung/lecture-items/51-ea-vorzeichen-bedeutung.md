# Bedeutung des Vorzeichens

## Negative EA: Stabiles Anion

Die meisten Atome geben Energie ab, wenn sie ein Elektron aufnehmen:

- Das Atom "will" das Elektron
- Das gebildete Anion ist **stabil**
- Je negativer der Wert, desto stabiler das Anion

**Beispiele:**

| Element | EA (kJ/mol) | Stabilität des Anions |
| ------- | ----------- | --------------------- |
| Cl      | −349        | Sehr stabil           |
| F       | −328        | Sehr stabil           |
| O       | −141        | Stabil                |
| S       | −200        | Stabil                |

## Positive EA: Ungünstige Aufnahme

Manche Atome nehmen Elektronen nur unter Zwang oder gar nicht auf:

- Das Atom "will" das Elektron nicht
- Energie muss zugeführt werden
- Das Anion wäre instabil

**Beispiele:**

| Element | EA (kJ/mol) | Grund               |
| ------- | ----------- | ------------------- |
| He      | +21         | Volle Schale        |
| Ne      | +29         | Volle Schale        |
| Be      | +19         | Volle s-Unterschale |

> **Merke:** Negative EA = "Elektronen-hungrig" (typisch für Nichtmetalle)
