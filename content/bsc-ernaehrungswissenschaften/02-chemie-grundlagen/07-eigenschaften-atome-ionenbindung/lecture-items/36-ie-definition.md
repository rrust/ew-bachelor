# Ionisierungsenergie – Definition

## Was ist Ionisierungsenergie?

Die **Ionisierungsenergie** (IE) ist ein Maß dafür, wie fest ein Atom seine Elektronen bindet.

> **Definition:** Die Ionisierungsenergie ist die minimal benötigte Energie, um das am schwächsten gebundene Elektron aus einem isolierten Atom im **Gaszustand** vollständig zu entfernen.

## Wichtige Bedingungen

1. Das Atom muss im **Gaszustand** vorliegen (isoliert, keine Wechselwirkungen)
2. Es wird das **schwächst gebundene** Elektron entfernt (das Valenzelektron)
3. Das Elektron wird **vollständig entfernt** (ins Unendliche)

## Notation

Die erste Ionisierungsenergie wird oft als $I_1$ oder IE₁ geschrieben:

$$I_1 = \text{Energie für } X_{(g)} \rightarrow X^+_{(g)} + e^-$$

## Einheit

Die Ionisierungsenergie wird üblicherweise in **kJ/mol** angegeben.

Beispiel Wasserstoff: $I_1 = 1312$ kJ/mol
