# IE-Trend in der Periode: Zunahme

## Der Trend

**Innerhalb einer Periode steigt die Ionisierungsenergie von links nach rechts.**

## Warum?

Zwei Faktoren wirken zusammen:

### 1. Steigende effektive Kernladung ($Z^*$)

- Mehr Protonen → stärkere Anziehung
- Die Valenzelektronen werden fester gehalten

### 2. Sinkender Atomradius

- Kleinere Atome → Elektronen näher am Kern
- Näher am Kern → stärkere Anziehung

## Die Konsequenz

Es wird von links nach rechts immer **schwerer**, ein Elektron zu entfernen!

## Beispiel: 2. Periode

| Element | $Z$ | 1. IE (kJ/mol) |
| ------- | --- | -------------- |
| Li      | 3   | 520            |
| Be      | 4   | 900            |
| B       | 5   | 801            |
| C       | 6   | 1086           |
| N       | 7   | 1402           |
| O       | 8   | 1314           |
| F       | 9   | 1681           |
| Ne      | 10  | 2081           |

> **Folge:** Edelgase haben die höchsten Ionisierungsenergien!
