question: 'Ein Element hat folgende Ionisierungsenergien: $I_1$ = 578, $I_2$ = 1817, $I_3$ = 2745, $I_4$ = 11577 kJ/mol. Welche Aussagen sind korrekt?'
options:
  - 'Das Element hat 3 Valenzelektronen'
  - 'Es gehört zur 3. Hauptgruppe'
  - 'Der große Sprung liegt zwischen $I_3$ und $I_4$'
  - 'Nach $I_3$ wird die Edelgaskonfiguration erreicht'
  - 'Das Element ist wahrscheinlich Aluminium'
  - 'Das Element ist wahrscheinlich Natrium'
correctAnswers:
  - 'Das Element hat 3 Valenzelektronen'
  - 'Es gehört zur 3. Hauptgruppe'
  - 'Der große Sprung liegt zwischen $I_3$ und $I_4$'
  - 'Nach $I_3$ wird die Edelgaskonfiguration erreicht'
  - 'Das Element ist wahrscheinlich Aluminium'
explanation: 'Der große Sprung nach $I_3$ zeigt 3 Valenzelektronen an (3. Hauptgruppe). Nach Entfernung der 3 Valenzelektronen hat Al³⁺ Neon-Konfiguration. Natrium hat nur 1 Valenzelektron.'
