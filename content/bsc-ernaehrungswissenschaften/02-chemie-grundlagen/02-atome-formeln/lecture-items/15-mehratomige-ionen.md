## Mehratomige Ionen (Molekülionen)

Neben einfachen Ionen wie $Na^+$ oder $Cl^-$ gibt es auch **mehratomige Ionen** – Gruppen von Atomen, die kovalent gebunden sind, aber insgesamt eine Ladung tragen.

### Wichtige Kationen

| Name     | Formel   | Ladung |
| -------- | -------- | ------ |
| Ammonium | $NH_4^+$ | +1     |

### Wichtige Anionen

Diese Ionen treten häufig in Verbindungen auf und sollten auswendig gelernt werden:

| Name             | Formel      | Ladung |
| ---------------- | ----------- | ------ |
| Hydroxid         | $OH^-$      | -1     |
| Nitrat           | $NO_3^-$    | -1     |
| Nitrit           | $NO_2^-$    | -1     |
| Carbonat         | $CO_3^{2-}$ | -2     |
| Hydrogencarbonat | $HCO_3^-$   | -1     |
| Sulfat           | $SO_4^{2-}$ | -2     |
| Sulfit           | $SO_3^{2-}$ | -2     |
| Phosphat         | $PO_4^{3-}$ | -3     |
| Acetat           | $CH_3COO^-$ | -1     |

### Systematik bei Sauerstoff-Ionen

Bei Oxoanionen (Sauerstoff enthaltend) gibt es ein Muster:

- **-at:** Mehr Sauerstoff (z.B. Nitrat $NO_3^-$, Sulfat $SO_4^{2-}$)
- **-it:** Weniger Sauerstoff (z.B. Nitrit $NO_2^-$, Sulfit $SO_3^{2-}$)

### Anwendung in Verbindungen

Mehratomige Ionen bleiben als **Einheit** erhalten. Bei Bedarf werden sie in Klammern gesetzt:

- Calciumnitrat: $Ca(NO_3)_2$ (ein $Ca^{2+}$ braucht zwei $NO_3^-$)
- Ammoniumsulfat: $(NH_4)_2SO_4$ (zwei $NH_4^+$ für ein $SO_4^{2-}$)
