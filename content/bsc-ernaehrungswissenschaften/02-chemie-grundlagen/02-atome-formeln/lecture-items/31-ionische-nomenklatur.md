## Nomenklatur ionischer Verbindungen

Um chemische Sachverhalte präzise zu kommunizieren, müssen Sie die systematischen Benennungsregeln (IUPAC) beherrschen.

### Grundregel

**Kation zuerst, Anion zuletzt.**

### 1. Metalle mit fester Ladung

Bei Hauptgruppenmetallen (Gruppe 1, 2, Aluminium) bleibt der Name unverändert:

| Formel    | Name            |
| --------- | --------------- |
| $NaCl$    | Natriumchlorid  |
| $CaO$     | Calciumoxid     |
| $Al_2O_3$ | Aluminiumoxid   |
| $MgBr_2$  | Magnesiumbromid |

### 2. Metalle mit variabler Ladung

Bei Übergangsmetallen **muss** die Ladung in römischen Ziffern angegeben werden (Stock-System):

| Formel   | Ion       | Name               |
| -------- | --------- | ------------------ |
| $FeCl_2$ | $Fe^{2+}$ | Eisen(II)-chlorid  |
| $FeCl_3$ | $Fe^{3+}$ | Eisen(III)-chlorid |
| $Cu_2O$  | $Cu^{+}$  | Kupfer(I)-oxid     |
| $CuO$    | $Cu^{2+}$ | Kupfer(II)-oxid    |

### 3. Anionen-Benennung

**Einatomige Anionen:** Endung **-id**
- $Cl^-$ → Chlorid
- $O^{2-}$ → Oxid
- $S^{2-}$ → Sulfid
- $N^{3-}$ → Nitrid

**Sauerstoffhaltige Anionen (Oxoanionen):**
- Mehr Sauerstoff: **-at** (Nitrat $NO_3^-$, Sulfat $SO_4^{2-}$)
- Weniger Sauerstoff: **-it** (Nitrit $NO_2^-$, Sulfit $SO_3^{2-}$)

### 4. Verbindungen mit mehratomigen Ionen

| Formel         | Name              |
| -------------- | ----------------- |
| $Ca(NO_3)_2$   | Calciumnitrat     |
| $(NH_4)_2SO_4$ | Ammoniumsulfat    |
| $Fe_2(SO_4)_3$ | Eisen(III)-sulfat |
| $K_2CO_3$      | Kaliumcarbonat    |
