# Umrechnungen: Masse, Mol und Teilchenzahl

## Die drei Größen

In der Chemie arbeiten wir mit drei zusammenhängenden Größen:

| Größe        | Symbol | Einheit | Bedeutung           |
| ------------ | ------ | ------- | ------------------- |
| Masse        | m      | g       | Wie viel Gramm?     |
| Stoffmenge   | n      | mol     | Wie viele "Pakete"? |
| Teilchenzahl | N      | -       | Wie viele Teilchen? |

## Die Grundformeln

### Masse ↔ Stoffmenge

$$n = \frac{m}{M}$$

$$m = n \times M$$

wobei M = molare Masse (g/mol)

### Stoffmenge ↔ Teilchenzahl

$$N = n \times N_A$$

$$n = \frac{N}{N_A}$$

wobei $N_A = 6{,}022 \times 10^{23}$ mol⁻¹

## Beispielrechnung

**Aufgabe:** Wie viele Wassermoleküle sind in 36 g Wasser?

**Schritt 1:** Stoffmenge berechnen
$$n = \frac{m}{M} = \frac{36 \text{ g}}{18 \text{ g/mol}} = 2 \text{ mol}$$

**Schritt 2:** Teilchenzahl berechnen
$$N = n \times N_A = 2 \times 6{,}022 \times 10^{23}$$
$$= 1{,}204 \times 10^{24} \text{ Moleküle}$$

## Typische Umrechnungsaufgaben

| Gegeben    | Gesucht      | Formel         |
| ---------- | ------------ | -------------- |
| Masse      | Stoffmenge   | n = m/M        |
| Stoffmenge | Masse        | m = n × M      |
| Stoffmenge | Teilchenzahl | N = n × Nₐ     |
| Masse      | Teilchenzahl | N = (m/M) × Nₐ |
