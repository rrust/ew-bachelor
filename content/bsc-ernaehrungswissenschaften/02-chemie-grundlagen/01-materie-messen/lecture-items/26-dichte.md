## Die Dichte

Die **Dichte** (ρ, "rho") ist eine intensive Eigenschaft – sie hängt nicht von der Stoffmenge ab, sondern ist charakteristisch für jeden Stoff.

### Definition und Formel

$$\rho = \frac{m}{V}$$

| Variable | Bedeutung | Einheit         |
| -------- | --------- | --------------- |
| $\rho$   | Dichte    | g/cm³ oder g/mL |
| $m$      | Masse     | g               |
| $V$      | Volumen   | cm³ oder mL     |

> **Hinweis:** $1\,\text{mL} = 1\,\text{cm}^3$

### Typische Dichten

| Stoff           | Dichte (g/cm³) |
| --------------- | -------------- |
| Luft (bei 20°C) | 0,0012         |
| Wasser          | 1,00           |
| Aluminium       | 2,70           |
| Eisen           | 7,87           |
| Quecksilber     | 13,6           |
| Gold            | 19,3           |
| Platin          | 21,5           |

### Anwendung: Schwimmen und Sinken

Ein Gegenstand schwimmt, wenn seine Dichte kleiner ist als die des umgebenden Mediums:
- Holz (ρ ≈ 0,5 g/cm³) schwimmt auf Wasser
- Eisen (ρ = 7,87 g/cm³) sinkt in Wasser
- Aluminium (ρ = 2,7 g/cm³) schwimmt auf Quecksilber

### Beispielrechnung

**Frage:** Welche Masse haben 95 mL Quecksilber?

$$m = \rho \times V = 13,6\,\frac{g}{cm^3} \times 95\,cm^3 = 1292\,g \approx 1,3\,kg$$
