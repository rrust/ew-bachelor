## Die drei Aggregatzustände

Materie besteht kinetisch betrachtet aus sich ständig bewegenden Atomen und Molekülen. Wir unterscheiden drei klassische **Aggregatzustände**:

### 1. Feststoffe (Solids)

| Eigenschaft       | Beschreibung                                     |
| ----------------- | ------------------------------------------------ |
| **Form**          | Starr, definierte Gestalt                        |
| **Volumen**       | Fest, ändert sich kaum                           |
| **Teilchen**      | In regelmäßigem Gitter angeordnet, vibrieren nur |
| **Komprimierbar** | Praktisch nicht                                  |

Die äußere Form von Kristallen spiegelt oft die innere, geordnete atomare Struktur wider.

### 2. Flüssigkeiten (Liquids)

| Eigenschaft       | Beschreibung                             |
| ----------------- | ---------------------------------------- |
| **Form**          | Keine eigene, passt sich dem Behälter an |
| **Volumen**       | Fest, definiert                          |
| **Teilchen**      | Beweglich, aber nah beieinander          |
| **Komprimierbar** | Kaum                                     |

Flüssigkeiten füllen den Behälter nicht vollständig – sie haben eine freie Oberfläche.

### 3. Gase (Gases)

| Eigenschaft       | Beschreibung                                    |
| ----------------- | ----------------------------------------------- |
| **Form**          | Keine eigene, füllt jeden Behälter vollständig  |
| **Volumen**       | Kein eigenes, dehnt sich aus                    |
| **Teilchen**      | Weit voneinander entfernt, bewegen sich schnell |
| **Komprimierbar** | Sehr gut                                        |

Theoretisch ist der gasförmige Zustand am besten verstanden.
