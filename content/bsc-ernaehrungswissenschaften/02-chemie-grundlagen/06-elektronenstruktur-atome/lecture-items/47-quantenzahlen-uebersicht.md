## Zusammenfassung: Die drei räumlichen Quantenzahlen

| Quantenzahl       | Symbol | Bestimmt               | Erlaubte Werte      |
| ----------------- | ------ | ---------------------- | ------------------- |
| Hauptquantenzahl  | n      | Schale, Energie, Größe | 1, 2, 3, 4, ...     |
| Nebenquantenzahl  | l      | Form (Unterschale)     | 0, 1, ..., n-1      |
| Magnetquantenzahl | mₗ     | Orientierung           | -l, ..., 0, ..., +l |

### Beispiel: Alle Orbitale der 3. Schale (n=3)

| Unterschale | l   | mₗ-Werte          | Orbitale                        | Anzahl         |
| ----------- | --- | ----------------- | ------------------------------- | -------------- |
| 3s          | 0   | 0                 | 3s                              | 1              |
| 3p          | 1   | -1, 0, +1         | 3px, 3py, 3pz                   | 3              |
| 3d          | 2   | -2, -1, 0, +1, +2 | 3dxy, 3dxz, 3dyz, 3dx²-y², 3dz² | 5              |
| **Gesamt**  |     |                   |                                 | **9 Orbitale** |

### Kontrolle: 9 Orbitale × 2 Elektronen = 18 Elektronen

Das passt zur Formel: $2n^2 = 2 \times 3^2 = 18$

### Die "Adresse" eines Orbitals

Ein Orbital ist durch (n, l, mₗ) eindeutig bestimmt:

- (2, 1, 0) → ein 2p-Orbital (das pz)
- (3, 2, -1) → eines der 3d-Orbitale
- (1, 0, 0) → das 1s-Orbital
