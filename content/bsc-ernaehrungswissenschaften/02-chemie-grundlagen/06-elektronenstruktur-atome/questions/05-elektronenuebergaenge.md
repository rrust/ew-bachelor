---
type: 'multiple-choice-multiple'
topic: 'Elektronenübergänge'
question: 'Welche der folgenden Elektronenübergänge im Wasserstoffatom sind mit Emission von Licht verbunden?'
options:
  - 'n = 3 → n = 1'
  - 'n = 1 → n = 4'
  - 'n = 5 → n = 2'
  - 'n = 2 → n = 3'
  - 'n = 4 → n = 2'
correctAnswers:
  - 'n = 3 → n = 1'
  - 'n = 5 → n = 2'
  - 'n = 4 → n = 2'
---

**Erklärung:** Bei Übergängen von höheren zu niedrigeren Energieniveaus (größeres n → kleineres n) wird Energie freigesetzt und als Photon emittiert. Bei Übergängen zu höheren Niveaus (n=1→n=4 oder n=2→n=3) muss Energie absorbiert werden.
