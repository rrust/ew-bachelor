## Bindungsordnung in der MO-Theorie

Die MO-Theorie ermöglicht eine quantitative Berechnung der Bindungsordnung:

### Formel

$$\text{Bindungsordnung} = \frac{1}{2} \left( n_{\text{bindend}} - n_{\text{antibindend}} \right)$$

wobei:
- $n_{\text{bindend}}$ = Anzahl der Elektronen in bindenden MOs
- $n_{\text{antibindend}}$ = Anzahl der Elektronen in antibindenden MOs

### Interpretation

| Bindungsordnung | Bedeutung                                     |
| :-------------: | :-------------------------------------------- |
|        0        | Keine stabile Bindung                         |
|        1        | Einfachbindung                                |
|        2        | Doppelbindung                                 |
|        3        | Dreifachbindung                               |
| 0,5 / 1,5 / ... | Bindung mit teilweise antibindendem Charakter |

### Beispiel: H₂

- 2 Elektronen im bindenden σ₁ₛ-MO
- 0 Elektronen im antibindenden σ₁ₛ*-MO
- BO = ½ × (2 - 0) = **1** (Einfachbindung)
