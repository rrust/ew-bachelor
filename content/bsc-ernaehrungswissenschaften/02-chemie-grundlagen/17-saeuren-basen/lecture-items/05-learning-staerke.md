# Stärke von Säuren und Basen

Die Stärke einer Säure oder Base hängt davon ab, wie vollständig sie in Wasser dissoziiert.

## Starke Säuren und Basen

**Starke Säuren** dissoziieren in Wasser zu **100%** – das Gleichgewicht liegt vollständig auf der Produktseite.

### Die wichtigsten starken Säuren

| Formel                  | Name          |
| ----------------------- | ------------- |
| $\text{HCl}$            | Salzsäure     |
| $\text{HNO}_3$          | Salpetersäure |
| $\text{H}_2\text{SO}_4$ | Schwefelsäure |
| $\text{HClO}_4$         | Perchlorsäure |

Reaktion (kein Gleichgewicht, sondern vollständige Reaktion):

$$\text{HCl(aq)} + \text{H}_2\text{O(l)} \rightarrow \text{H}_3\text{O}^+\text{(aq)} + \text{Cl}^-\text{(aq)}$$

**Starke Basen** sind meist Hydroxide der Alkali- und Erdalkalimetalle:

| Formel            | Name            |
| ----------------- | --------------- |
| $\text{NaOH}$     | Natriumhydroxid |
| $\text{KOH}$      | Kaliumhydroxid  |
| $\text{Ca(OH)}_2$ | Calciumhydroxid |

## Schwache Säuren und Basen

**Schwache Säuren** dissoziieren nur teilweise (oft < 5%). Es stellt sich ein Gleichgewicht ein:

$$\text{CH}_3\text{COOH(aq)} + \text{H}_2\text{O(l)} \rightleftharpoons \text{CH}_3\text{COO}^-\text{(aq)} + \text{H}_3\text{O}^+\text{(aq)}$$

### Beispiele schwacher Säuren

| Formel                   | Name        |
| ------------------------ | ----------- |
| $\text{CH}_3\text{COOH}$ | Essigsäure  |
| $\text{HF}$              | Flusssäure  |
| $\text{HCN}$             | Blausäure   |
| $\text{H}_2\text{CO}_3$  | Kohlensäure |

## Der Zusammenhang der Stärke

> **Wichtige Regel:** Je stärker eine Säure, desto schwächer ihre konjugierte Base – und umgekehrt!

| Säure                | Stärke       | Konjugierte Base | Stärke         |
| -------------------- | ------------ | ---------------- | -------------- |
| $\text{HCl}$         | sehr stark   | $\text{Cl}^-$    | extrem schwach |
| $\text{HF}$          | schwach      | $\text{F}^-$     | mäßig stark    |
| $\text{H}_2\text{O}$ | sehr schwach | $\text{OH}^-$    | stark          |

Säure-Base-Reaktionen verlaufen immer vom **stärkeren** zum **schwächeren** Paar.
