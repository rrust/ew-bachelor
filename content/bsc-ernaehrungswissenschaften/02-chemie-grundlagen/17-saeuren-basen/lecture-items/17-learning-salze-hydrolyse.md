# Salze und Hydrolyse

Wenn Salze in Wasser gelöst werden, können sie den pH-Wert beeinflussen. Diese Reaktion von Ionen mit Wasser nennt man **Hydrolyse**.

## Die vier Typen von Salzlösungen

### Typ 1: Starke Base + Starke Säure → Neutral

Beispiele: $\text{NaCl}$, $\text{KNO}_3$

- **pH ≈ 7** (neutral)
- Weder Kation noch Anion reagieren mit Wasser
- $\text{Na}^+$ und $\text{Cl}^-$ sind spektator-Ionen

### Typ 2: Starke Base + Schwache Säure → Basisch

Beispiele: $\text{NaCH}_3\text{COO}$ (Natriumacetat), $\text{NaClO}$

- **pH > 7** (basisch)
- Das Anion ist die konjugierte Base einer schwachen Säure
- Hydrolyse-Reaktion:

$$\text{A}^- + \text{H}_2\text{O} \rightleftharpoons \text{HA} + \text{OH}^-$$

### Typ 3: Schwache Base + Starke Säure → Sauer

Beispiele: $\text{NH}_4\text{Cl}$ (Ammoniumchlorid)

- **pH < 7** (sauer)
- Das Kation ist die konjugierte Säure einer schwachen Base
- Hydrolyse-Reaktion:

$$\text{BH}^+ + \text{H}_2\text{O} \rightleftharpoons \text{B} + \text{H}_3\text{O}^+$$

### Typ 4: Schwache Base + Schwache Säure → Kommt drauf an

Beispiel: $\text{NH}_4\text{CH}_3\text{COO}$ (Ammoniumacetat)

Der pH hängt vom Verhältnis der Konstanten ab:

| Bedingung                                | Ergebnis |
| ---------------------------------------- | -------- |
| $K_s(\text{Kation}) > K_b(\text{Anion})$ | sauer    |
| $K_b(\text{Anion}) > K_s(\text{Kation})$ | basisch  |
| $K_s \approx K_b$                        | neutral  |

## Zusammenfassung

| Salz-Typ  | Kation von    | Anion von      | pH           |
| --------- | ------------- | -------------- | ------------ |
| NaCl      | starke Base   | starke Säure   | 7            |
| NaCH₃COO  | starke Base   | schwache Säure | > 7          |
| NH₄Cl     | schwache Base | starke Säure   | < 7          |
| NH₄CH₃COO | schwache Base | schwache Säure | vergleichen! |
