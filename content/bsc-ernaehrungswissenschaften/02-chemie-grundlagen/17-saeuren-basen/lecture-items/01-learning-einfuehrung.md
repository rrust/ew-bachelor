# Säuren und Basen – Einführung

Säuren und Basen begegnen uns überall im Alltag: Der saure Geschmack von Zitronen stammt von der **Zitronensäure**, der Schmerz eines Ameisenbisses wird durch **Ameisensäure** ($\text{HCO}_2\text{H}$) verursacht.

## Brønsted-Lowry Theorie

Die gebräuchlichste Definition von Säuren und Basen stammt von Brønsted und Lowry:

| Begriff   | Definition                                |
| --------- | ----------------------------------------- |
| **Säure** | Protonendonator – gibt $\text{H}^+$ ab    |
| **Base**  | Protonenakzeptor – nimmt $\text{H}^+$ auf |

## Konjugierte Säure-Base-Paare

Bei jeder Säure-Base-Reaktion entsteht ein **konjugiertes Paar**:

$$\text{Säure} + \text{Base} \rightleftharpoons \text{konj. Base} + \text{konj. Säure}$$

### Beispiel: Ammoniak in Wasser

$$\text{NH}_3\text{(aq)} + \text{H}_2\text{O(l)} \rightleftharpoons \text{NH}_4^+\text{(aq)} + \text{OH}^-\text{(aq)}$$

| Rolle             | Stoff                | Erklärung                        |
| ----------------- | -------------------- | -------------------------------- |
| Base              | $\text{NH}_3$        | Nimmt $\text{H}^+$ auf           |
| Konjugierte Säure | $\text{NH}_4^+$      | Kann $\text{H}^+$ wieder abgeben |
| Säure             | $\text{H}_2\text{O}$ | Gibt $\text{H}^+$ ab             |
| Konjugierte Base  | $\text{OH}^-$        | Kann $\text{H}^+$ aufnehmen      |

## Merke

Jede Säure hat eine konjugierte Base und umgekehrt. Sie unterscheiden sich nur durch **ein einziges Proton** ($\text{H}^+$).

| Säure                   | Konjugierte Base |
| ----------------------- | ---------------- |
| $\text{HCl}$            | $\text{Cl}^-$    |
| $\text{H}_2\text{SO}_4$ | $\text{HSO}_4^-$ |
| $\text{NH}_4^+$         | $\text{NH}_3$    |
| $\text{H}_2\text{O}$    | $\text{OH}^-$    |
