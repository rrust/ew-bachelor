# Schrittweise Anleitung zum Ausgleichen

Das Ausgleichen von Reaktionsgleichungen ist eine wichtige Grundfertigkeit in der Chemie.

## Die Methode

### Schritt 1: Gleichung aufschreiben

Schreiben Sie die unausgeglichene Gleichung mit korrekten Formeln auf.

### Schritt 2: Atome zählen

Zählen Sie jeden Atomtyp auf beiden Seiten.

### Schritt 3: Koeffizienten anpassen

Beginnen Sie mit dem komplexesten Molekül und passen Sie die Koeffizienten an.

### Schritt 4: Kontrolle

Prüfen Sie, ob alle Atome ausgeglichen sind.

## Praktische Tipps

1. **Beginnen Sie mit Metallen** – sie kommen oft nur in einem Molekül vor
2. **Sauerstoff zuletzt** – O₂ lässt sich leicht anpassen
3. **Bei Brüchen** – alle Koeffizienten verdoppeln
4. **Immer kontrollieren** – Atome auf beiden Seiten zählen

## Beispiel: Eisen und Chlor

**Unausgeglichen:** Fe + Cl₂ → FeCl₃

| Schritt     | Gleichung                  | Fe    | Cl    |
| ----------- | -------------------------- | ----- | ----- |
| Start       | Fe + Cl₂ → FeCl₃           | 1:1 ✓ | 2:3 ❌ |
| Cl anpassen | Fe + 3/2 Cl₂ → FeCl₃       | 1:1   | 3:3 ✓ |
| Verdoppeln  | **2 Fe + 3 Cl₂ → 2 FeCl₃** | 2:2 ✓ | 6:6 ✓ |

> ⚠️ **Wichtig:** Vermeiden Sie Brüche in der finalen Gleichung – verdoppeln Sie alle Koeffizienten!
