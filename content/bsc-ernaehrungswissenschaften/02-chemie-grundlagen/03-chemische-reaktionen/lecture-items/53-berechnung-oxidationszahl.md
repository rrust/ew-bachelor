question: 'Bestimmen Sie die Oxidationszahl von Stickstoff in folgenden Verbindungen und berechnen Sie die Summe aller drei Oxidationszahlen.'
task: |
  Bestimmen Sie die Oxidationszahl von Stickstoff (N) in:
  
  1. **NH₃** (Ammoniak)
  2. **NO₂** (Stickstoffdioxid)  
  3. **HNO₃** (Salpetersäure)
  
  Berechnen Sie: OZ(N in NH₃) + OZ(N in NO₂) + OZ(N in HNO₃) = ?
correctAnswer: '6'
unit: ''
solution: |
  **1. NH₃ (Ammoniak)**
  - H = +1 (Regel)
  - 3 × (+1) + x = 0
  - x = **−3**
  
  **2. NO₂ (Stickstoffdioxid)**
  - O = −2 (Regel)
  - x + 2 × (−2) = 0
  - x = **+4**
  
  **3. HNO₃ (Salpetersäure)**
  - H = +1, O = −2
  - (+1) + x + 3 × (−2) = 0
  - +1 + x − 6 = 0
  - x = **+5**
  
  **Summe:** (−3) + (+4) + (+5) = **+6**
hint: 'Wenden Sie die Regeln an: H = +1, O = −2, Summe = 0 (bei neutralen Molekülen).'
