# Löslichkeitsregeln

Um vorherzusagen, ob ein Niederschlag entsteht, müssen wir wissen, welche Salze löslich und welche schwerlöslich sind.

## Die wichtigsten Regeln

### Immer löslich ✓

| Ionentyp                        | Beispiele              |
| ------------------------------- | ---------------------- |
| **Nitrate** (NO₃⁻)              | NaNO₃, AgNO₃, Pb(NO₃)₂ |
| **Alkalimetallsalze** (Na⁺, K⁺) | Na₂SO₄, K₂CO₃, NaCl    |
| **Ammoniumsalze** (NH₄⁺)        | NH₄Cl, (NH₄)₂SO₄       |

### Meist löslich (mit Ausnahmen)

| Ionentyp            | Ausnahmen (schwerlöslich) |
| ------------------- | ------------------------- |
| **Chloride** (Cl⁻)  | AgCl, PbCl₂, Hg₂Cl₂       |
| **Sulfate** (SO₄²⁻) | BaSO₄, PbSO₄, CaSO₄       |

### Meist schwerlöslich

| Ionentyp              | Ausnahmen (löslich)       |
| --------------------- | ------------------------- |
| **Carbonate** (CO₃²⁻) | Alkali- und Ammoniumsalze |
| **Phosphate** (PO₄³⁻) | Alkali- und Ammoniumsalze |
| **Hydroxide** (OH⁻)   | NaOH, KOH, Ba(OH)₂        |
| **Sulfide** (S²⁻)     | Alkali- und Ammoniumsalze |

## Merkhilfe

> **Fast alle Nitrate und Alkalimetallsalze sind löslich!**

> **Die meisten Carbonate, Phosphate und Hydroxide sind schwerlöslich** (außer mit Alkali/Ammonium).

## Beispiele aus der Natur

- **Kalkstein** (CaCO₃) – schwerlösliches Carbonat
- **Eisenkies** (FeS₂) – schwerlösliches Sulfid
- **Gips** (CaSO₄) – schwerlösliches Sulfat
