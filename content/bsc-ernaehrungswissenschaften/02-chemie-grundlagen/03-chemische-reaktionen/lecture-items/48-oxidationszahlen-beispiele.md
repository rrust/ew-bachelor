# Oxidationszahlen berechnen

## Beispiel 1: Schwefelsäure (H₂SO₄)

**Gegeben:**
- H = +1 (Regel)
- O = −2 (Regel)
- Summe = 0 (neutral)

**Berechnung:**
$$2 \cdot (+1) + x + 4 \cdot (-2) = 0$$
$$+2 + x - 8 = 0$$
$$x = +6$$

**Ergebnis:** Schwefel hat OZ = **+6**

## Beispiel 2: Permanganat-Ion (MnO₄⁻)

**Gegeben:**
- O = −2
- Summe = −1 (Ionenladung)

**Berechnung:**
$$x + 4 \cdot (-2) = -1$$
$$x - 8 = -1$$
$$x = +7$$

**Ergebnis:** Mangan hat OZ = **+7**

## Beispiel 3: Dichromat-Ion (Cr₂O₇²⁻)

**Gegeben:**
- O = −2
- Summe = −2

**Berechnung:**
$$2x + 7 \cdot (-2) = -2$$
$$2x - 14 = -2$$
$$2x = +12$$
$$x = +6$$

**Ergebnis:** Jedes Chrom hat OZ = **+6**

## Übersicht häufiger Oxidationszahlen

| Element          | Häufige OZ                |
| ---------------- | ------------------------- |
| O                | −2 (−1 in Peroxiden)      |
| H                | +1 (−1 in Hydriden)       |
| Alkalimetalle    | +1                        |
| Erdalkalimetalle | +2                        |
| Halogene         | −1 (in Ionenverbindungen) |
| N                | −3 bis +5                 |
| S                | −2 bis +6                 |
| C                | −4 bis +4                 |
