  Betrachten Sie die Dissoziation von Kupfer(II)-chlorid in Wasser:
  
  CuCl₂(s) → Cu²⁺(aq) + 2 Cl⁻(aq)
  
  Welche Aussagen sind korrekt?
options:
  - 'CuCl₂ ist ein starker Elektrolyt'
  - 'Der einfache Pfeil (→) zeigt vollständige Dissoziation an'
  - 'Pro Formeleinheit CuCl₂ entstehen 3 Ionen'
  - 'Die Lösung leitet elektrischen Strom'
  - 'Kupfer liegt als Cu⁺-Ion vor'
correctAnswers:
  - 'CuCl₂ ist ein starker Elektrolyt'
  - 'Der einfache Pfeil (→) zeigt vollständige Dissoziation an'
  - 'Pro Formeleinheit CuCl₂ entstehen 3 Ionen'
  - 'Die Lösung leitet elektrischen Strom'
hint: 'Zählen Sie die Ionen: 1 Cu²⁺ + 2 Cl⁻. Welche Ladung hat Kupfer in CuCl₂?'
explanation: |
  - ✓ CuCl₂ ist ein Salz → starker Elektrolyt
  - ✓ Einfacher Pfeil = vollständig
  - ✓ 1 Cu²⁺ + 2 Cl⁻ = 3 Ionen
  - ✓ Ionen in Lösung → Stromleitung
  - ✗ Kupfer ist **Cu²⁺** (Kupfer-II), nicht Cu⁺
