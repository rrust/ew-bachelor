type: 'multiple-choice-multiple'
question: |
  Betrachten Sie die Reaktion: Zn(s) + Cu²⁺(aq) → Zn²⁺(aq) + Cu(s)
  
  Welche Aussagen sind korrekt?
options:
  - 'Zink wird oxidiert'
  - 'Kupfer-Ionen werden reduziert'
  - 'Zink ist das Reduktionsmittel'
  - 'Cu²⁺ ist das Reduktionsmittel'
  - 'Die Oxidationszahl von Zink steigt von 0 auf +2'
correctAnswers:
  - 'Zink wird oxidiert'
  - 'Kupfer-Ionen werden reduziert'
  - 'Zink ist das Reduktionsmittel'
  - 'Die Oxidationszahl von Zink steigt von 0 auf +2'
hint: 'Zink gibt Elektronen ab (wird oxidiert) und reduziert damit das Kupfer. Wer gibt, ist das Reduktionsmittel.'
explanation: |
  - ✓ Zn → Zn²⁺ + 2 e⁻ = Oxidation
  - ✓ Cu²⁺ + 2 e⁻ → Cu = Reduktion
  - ✓ Zink gibt e⁻ ab → **Reduktionsmittel**
  - ✗ Cu²⁺ nimmt e⁻ auf → **Oxidationsmittel** (nicht Reduktionsmittel!)
  - ✓ OZ Zn: 0 → +2 (steigt)
