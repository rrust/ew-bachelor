type: 'multiple-choice-multiple'
question: |
  Bestimmen Sie die Oxidationszahlen in verschiedenen Verbindungen.
  
  Welche Aussagen sind korrekt?
options:
  - 'Schwefel in H₂SO₄ hat die Oxidationszahl +6'
  - 'Stickstoff in NH₃ hat die Oxidationszahl −3'
  - 'Mangan in MnO₄⁻ hat die Oxidationszahl +7'
  - 'Sauerstoff hat immer die Oxidationszahl −2'
  - 'Chrom in Cr₂O₇²⁻ hat die Oxidationszahl +6'
correctAnswers:
  - 'Schwefel in H₂SO₄ hat die Oxidationszahl +6'
  - 'Stickstoff in NH₃ hat die Oxidationszahl −3'
  - 'Mangan in MnO₄⁻ hat die Oxidationszahl +7'
  - 'Chrom in Cr₂O₇²⁻ hat die Oxidationszahl +6'
hint: 'Berechnen Sie: H = +1, O = −2, Summe = 0 (neutral) oder Ionenladung.'
explanation: |
  Berechnungen:
  - H₂SO₄: 2(+1) + x + 4(−2) = 0 → x = **+6** ✓
  - NH₃: x + 3(+1) = 0 → x = **−3** ✓
  - MnO₄⁻: x + 4(−2) = −1 → x = **+7** ✓
  - ✗ Sauerstoff ist meist −2, aber **nicht immer** (z.B. −1 in Peroxiden)
  - Cr₂O₇²⁻: 2x + 7(−2) = −2 → x = **+6** ✓
