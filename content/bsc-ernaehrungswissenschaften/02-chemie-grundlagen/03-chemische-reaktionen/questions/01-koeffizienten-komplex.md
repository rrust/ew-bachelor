type: 'multiple-choice-multiple'
question: |
  Betrachten Sie die Reaktionsgleichung für die Verbrennung von Ethanol:
  
  C₂H₅OH + __ O₂ → __ CO₂ + __ H₂O
  
  Welche Aussagen sind korrekt?
options:
  - 'Der Koeffizient vor O₂ ist 3'
  - 'Es entstehen 2 Moleküle CO₂ pro Molekül Ethanol'
  - 'Es entstehen 3 Moleküle H₂O pro Molekül Ethanol'
  - 'Die Summe aller Koeffizienten ist 9'
  - 'Ethanol ist das Oxidationsmittel in dieser Reaktion'
correctAnswers:
  - 'Der Koeffizient vor O₂ ist 3'
  - 'Es entstehen 2 Moleküle CO₂ pro Molekül Ethanol'
  - 'Es entstehen 3 Moleküle H₂O pro Molekül Ethanol'
hint: 'Gleichen Sie zuerst C, dann H, dann O aus. Die ausgeglichene Gleichung lautet: C₂H₅OH + 3 O₂ → 2 CO₂ + 3 H₂O'
explanation: |
  **Ausgeglichene Gleichung:** C₂H₅OH + 3 O₂ → 2 CO₂ + 3 H₂O
  
  - ✓ Koeffizient vor O₂ = 3
  - ✓ Es entstehen 2 CO₂
  - ✓ Es entstehen 3 H₂O
  - ✗ Summe: 1 + 3 + 2 + 3 = 9 wäre richtig, ABER Ethanol hat keinen expliziten Koeffizienten "1" in der Summenangabe
  - ✗ Ethanol ist das **Reduktionsmittel** (wird oxidiert), O₂ ist das Oxidationsmittel
