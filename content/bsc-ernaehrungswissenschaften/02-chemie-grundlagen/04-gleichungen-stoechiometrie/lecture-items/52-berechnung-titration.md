question: 'Berechnen Sie die Konzentration aus Titrationsdaten.'
task: |
  **Titrationsanalyse**
  
  25,0 mL einer Essigsäure-Lösung (CH₃COOH, monoprotisch) werden mit 0,1 M NaOH titriert.
  
  Bis zum Farbumschlag werden **18,5 mL NaOH** verbraucht.
  
  Reaktion: CH₃COOH + NaOH → CH₃COONa + H₂O (1:1)
  
  **Berechnen Sie die Konzentration der Essigsäure in mol/L.**
correctAnswer: '0,074'
unit: 'mol/L'
solution: |
  **Schritt 1: Mol NaOH berechnen**
  
  $$n_{NaOH} = c \times V = 0,1\ mol/L \times 0,0185\ L = 0,00185\ mol$$
  
  **Schritt 2: Mol Essigsäure (1:1 Verhältnis)**
  
  $$n_{CH_3COOH} = n_{NaOH} = 0,00185\ mol$$
  
  **Schritt 3: Konzentration berechnen**
  
  $$c_{CH_3COOH} = \frac{n}{V} = \frac{0,00185\ mol}{0,025\ L} = 0,074\ mol/L$$
  
  **Ergebnis:** Die Essigsäure-Konzentration beträgt **0,074 M** (oder 74 mM).
hint: 'Berechnen Sie zuerst die Stoffmenge NaOH (n = c × V), dann nutzen Sie das 1:1 Verhältnis, dann c = n/V für die Essigsäure.'
