# Analyse durch Fällung (Gravimetrie)

Die **Gravimetrie** ist eine analytische Methode, bei der ein Stoff durch Fällung bestimmt wird.

## Prinzip

1. Der gesuchte Stoff wird in eine **unlösliche Verbindung** überführt (Niederschlag)
2. Der Niederschlag wird **gefiltert, gewaschen und gewogen**
3. Aus der Masse wird auf die ursprüngliche Stoffmenge zurückgerechnet

## Beispiel: Sulfatbestimmung

**Fragestellung:** Wie viel Na₂SO₄ enthält eine Mineralprobe?

**Reaktion:**
$$Na_2SO_4(aq) + BaCl_2(aq) \rightarrow BaSO_4(s) + 2\ NaCl(aq)$$

Das Sulfat wird als **Bariumsulfat** gefällt – ein schwerlöslicher weißer Niederschlag.

## Berechnung

Wenn 1,0882 g BaSO₄ gewogen wurden:

**Schritt 1: Mol BaSO₄**
$$n_{BaSO_4} = \frac{1,0882\ g}{233,4\ g/mol} = 0,00466\ mol$$

**Schritt 2: Mol Na₂SO₄** (Stöchiometrie 1:1)
$$n_{Na_2SO_4} = 0,00466\ mol$$

**Schritt 3: Masse Na₂SO₄**
$$m_{Na_2SO_4} = 0,00466\ mol \times 142,0\ g/mol = 0,662\ g$$

## Vorteile der Gravimetrie

| Vorteil              | Erklärung                  |
| -------------------- | -------------------------- |
| **Hohe Genauigkeit** | Waagen sind sehr präzise   |
| **Einfache Geräte**  | Nur Filter und Waage nötig |
| **Rückführbarkeit**  | Masse ist eine Grundgröße  |
