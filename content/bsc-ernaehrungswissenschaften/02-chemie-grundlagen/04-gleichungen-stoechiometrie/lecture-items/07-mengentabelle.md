# Die Mengentabelle (Bilanzierung)

Eine **Mengentabelle** hilft, die Übersicht zu behalten und die Massenerhaltung zu prüfen.

## Aufbau einer Mengentabelle

Am Beispiel: $NH_4NO_3(s) \rightarrow N_2O(g) + 2\ H_2O(g)$

| Zeile                 | NH₄NO₃   | N₂O   | H₂O     |
| --------------------- | -------- | ----- | ------- |
| **Anfangs (g)**       | 454 g    | 0     | 0       |
| **Anfangs (mol)**     | 5,67 mol | 0     | 0       |
| **Veränderung (mol)** | −5,67    | +5,67 | +2×5,67 |
| **Final (mol)**       | 0        | 5,67  | 11,34   |
| **Final (g)**         | 0        | 250 g | 204 g   |

## Vorteile der Mengentabelle

1. **Übersichtlichkeit:** Alle Werte auf einen Blick
2. **Fehlerkontrolle:** Massenerhaltung leicht prüfbar
3. **Systematik:** Funktioniert für jede Reaktion

## Vorzeichen-Konvention

| Vorzeichen    | Bedeutung                         |
| ------------- | --------------------------------- |
| **−** (Minus) | Stoff wird **verbraucht** (Edukt) |
| **+** (Plus)  | Stoff **entsteht** (Produkt)      |

## Wichtige Beziehung

Die Veränderungen stehen im **Verhältnis der Koeffizienten**:

$$\frac{\Delta n_A}{\text{Koeff}_A} = \frac{\Delta n_B}{\text{Koeff}_B}$$

Im Beispiel:
- NH₄NO₃ hat Koeffizient 1 → Δn = −5,67
- H₂O hat Koeffizient 2 → Δn = +2 × 5,67 = +11,34
