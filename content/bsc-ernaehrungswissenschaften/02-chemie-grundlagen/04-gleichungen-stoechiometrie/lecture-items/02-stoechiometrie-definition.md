# Was ist Stöchiometrie?

Die **Stöchiometrie** ist die Untersuchung der quantitativen Aspekte chemischer Reaktionen. Sie bildet das mathematische Fundament der Chemie.

## Das Gesetz der Massenerhaltung

Das zugrundeliegende Prinzip ist das **Gesetz der Massenerhaltung** (auch: Lavoisier-Gesetz):

> **In einer chemischen Reaktion kann Materie weder geschaffen noch vernichtet werden.**

Die Masse der Edukte (Reaktanten) muss gleich der Masse der Produkte sein.

## Beispielreaktion

Betrachten wir die Reaktion von Aluminium mit Brom:

$$2\ Al(s) + 3\ Br_2(l) \rightarrow Al_2Br_6(s)$$

**Was sagt uns diese Gleichung?**

| Komponente | Bedeutung                      |
| ---------- | ------------------------------ |
| **2 Al**   | 2 Atome Aluminium              |
| **3 Br₂**  | 3 Moleküle Brom (= 6 Br-Atome) |
| **Al₂Br₆** | 1 Molekül Aluminiumbromid      |

## Atomzählung (Massenerhaltung prüfen)

| Element | Links (Edukte) | Rechts (Produkte) |
| ------- | -------------- | ----------------- |
| **Al**  | 2              | 2 ✓               |
| **Br**  | 3 × 2 = 6      | 6 ✓               |

Die Anzahl der Atome jedes Elements ist auf **beiden Seiten gleich** – das ist das Wesen der Massenerhaltung!

## Die Koeffizienten

Die Zahlen **vor** den chemischen Formeln heißen **Koeffizienten** (oder stöchiometrische Faktoren):
- Sie geben das **Mol-Verhältnis** der Reaktionspartner an
- Sie dürfen beim Ausgleichen verändert werden
- Die **Indizes** (kleine Zahlen in den Formeln) dürfen NICHT verändert werden!
