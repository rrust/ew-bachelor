# Titration: Konzentrationsbestimmung

Die **Titration** ist eine analytische Methode zur genauen Bestimmung der Konzentration einer Lösung.

## Prinzip

1. Eine Lösung **unbekannter Konzentration** (Analyt) wird vorgelegt
2. Eine Lösung **bekannter Konzentration** (Titrator) wird langsam zugegeben
3. Am **Äquivalenzpunkt** hat genau die stöchiometrische Menge reagiert
4. Der Äquivalenzpunkt wird durch einen **Indikator** (Farbumschlag) angezeigt

## Aufbau

| Bestandteil          | Funktion                                           |
| -------------------- | -------------------------------------------------- |
| **Bürette**          | Enthält Titrator, ermöglicht genaue Volumenmessung |
| **Erlenmeyerkolben** | Enthält Analyt (Probe)                             |
| **Indikator**        | Zeigt Äquivalenzpunkt durch Farbwechsel an         |

## Der Äquivalenzpunkt

Am Äquivalenzpunkt sind **Stoffmenge Säure** und **Stoffmenge Base** im exakten stöchiometrischen Verhältnis:

$$n_{Säure} \times z_{Säure} = n_{Base} \times z_{Base}$$

wobei $z$ die Anzahl der übertragbaren H⁺ bzw. OH⁻ ist.

## Typische Indikatoren

| Indikator           | Farbumschlag   | pH-Bereich |
| ------------------- | -------------- | ---------- |
| **Phenolphthalein** | farblos → rosa | 8,2 - 10   |
| **Methylorange**    | rot → gelb     | 3,1 - 4,4  |
| **Bromthymolblau**  | gelb → blau    | 6,0 - 7,6  |
