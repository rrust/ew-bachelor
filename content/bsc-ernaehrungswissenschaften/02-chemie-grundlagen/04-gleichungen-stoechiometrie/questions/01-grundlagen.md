  Welche Aussagen zur Stöchiometrie und zum Gesetz der Massenerhaltung sind korrekt?
options:
  - 'Die Koeffizienten in einer chemischen Gleichung geben das Molverhältnis an'
  - 'Die Gesamtmasse der Produkte muss der Gesamtmasse der Edukte entsprechen'
  - 'Bei einer Verbrennung geht Masse verloren, weil Gase entstehen'
  - 'Der stöchiometrische Faktor wird aus den Koeffizienten berechnet'
  - 'In einer ausgeglichenen Gleichung muss die Anzahl der Atome jedes Elements gleich sein'
correctAnswers:
  - 'Die Koeffizienten in einer chemischen Gleichung geben das Molverhältnis an'
  - 'Die Gesamtmasse der Produkte muss der Gesamtmasse der Edukte entsprechen'
  - 'Der stöchiometrische Faktor wird aus den Koeffizienten berechnet'
  - 'In einer ausgeglichenen Gleichung muss die Anzahl der Atome jedes Elements gleich sein'
explanation: 'Bei Verbrennungen geht KEINE Masse verloren - die Gase haben Masse! Sie entweichen nur aus dem Reaktionsgefäß, weshalb es so erscheint.'
