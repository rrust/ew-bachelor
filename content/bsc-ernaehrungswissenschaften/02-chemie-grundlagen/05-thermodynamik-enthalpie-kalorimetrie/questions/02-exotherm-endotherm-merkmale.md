question: |
  Welche der folgenden Aussagen zu exothermen und endothermen Reaktionen sind korrekt?
options:
  - 'Bei exothermen Reaktionen wird Energie an die Umgebung abgegeben'
  - 'Bei exothermen Reaktionen ist ΔH negativ'
  - 'Bei endothermen Reaktionen fühlt sich das Reaktionsgefäß warm an'
  - 'Die Verdampfung von Wasser ist ein endothermer Prozess'
  - 'Die Verbrennung von Methan ist ein endothermer Prozess'
correctAnswers:
  - 'Bei exothermen Reaktionen wird Energie an die Umgebung abgegeben'
  - 'Bei exothermen Reaktionen ist ΔH negativ'
  - 'Die Verdampfung von Wasser ist ein endothermer Prozess'
hint: 'Exotherm = Energie raus = ΔH < 0 = Gefäß wird warm. Endotherm = Energie rein = ΔH > 0 = Gefäß wird kalt.'
explanation: 'Exotherme Reaktionen geben Energie ab (ΔH < 0), das Gefäß wird warm. Endotherme Reaktionen nehmen Energie auf (ΔH > 0), das Gefäß wird kalt. Verdampfung ist endotherm (Energie nötig), Verbrennung ist exotherm (Energie frei).'
