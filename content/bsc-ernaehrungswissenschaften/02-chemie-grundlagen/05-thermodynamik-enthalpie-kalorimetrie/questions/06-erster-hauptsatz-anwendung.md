question: |
  Für den Ersten Hauptsatz der Thermodynamik ($\Delta U = q + w$) gelten bestimmte Vorzeichenkonventionen.
  
  Welche Aussagen sind korrekt?
options:
  - 'Wenn ein Gas expandiert, ist w negativ (System leistet Arbeit)'
  - 'Wenn Wärme ins System fließt, ist q positiv'
  - 'Bei adiabatischer Kompression ist q = 0 und w > 0'
  - 'Bei einer exothermen Reaktion ist q immer positiv'
  - 'In einem isolierten System gilt ΔU = 0'
correctAnswers:
  - 'Wenn ein Gas expandiert, ist w negativ (System leistet Arbeit)'
  - 'Wenn Wärme ins System fließt, ist q positiv'
  - 'Bei adiabatischer Kompression ist q = 0 und w > 0'
hint: 'Die Konvention ist: Energie ins System = positiv, Energie aus dem System = negativ.'
explanation: 'Die Vorzeichenkonvention: Energie INS System = positiv (+), Energie RAUS = negativ (-). Expansion: System gibt Energie ab → w < 0. Kompression: Energie wird zugeführt → w > 0. Adiabatisch bedeutet q = 0. Bei exothermen Reaktionen fließt Wärme RAUS → q < 0 (nicht positiv!).'
