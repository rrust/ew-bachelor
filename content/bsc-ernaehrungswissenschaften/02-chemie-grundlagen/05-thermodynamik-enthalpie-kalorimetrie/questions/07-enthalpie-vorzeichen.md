question: |
  Betrachte das Vorzeichen von $\Delta H$ für verschiedene Reaktionen.
  
  Welche Zuordnungen sind korrekt?
options:
  - 'Verbrennung von Glucose: ΔH < 0'
  - 'Schmelzen von Eis: ΔH > 0'
  - 'Photosynthese: ΔH < 0'
  - 'Neutralisation von HCl mit NaOH: ΔH < 0'
  - 'Lösen von Ammoniumnitrat in Wasser: ΔH > 0'
correctAnswers:
  - 'Verbrennung von Glucose: ΔH < 0'
  - 'Schmelzen von Eis: ΔH > 0'
  - 'Neutralisation von HCl mit NaOH: ΔH < 0'
  - 'Lösen von Ammoniumnitrat in Wasser: ΔH > 0'
hint: 'Exotherm (Wärme wird frei) = ΔH < 0, Endotherm (Wärme wird aufgenommen) = ΔH > 0.'
explanation: 'Verbrennung und Neutralisation sind exotherm (ΔH < 0). Schmelzen und Lösen von NH₄NO₃ sind endotherm (ΔH > 0). Photosynthese speichert Energie in Glucose → ΔH > 0 (endotherm), nicht exotherm!'
