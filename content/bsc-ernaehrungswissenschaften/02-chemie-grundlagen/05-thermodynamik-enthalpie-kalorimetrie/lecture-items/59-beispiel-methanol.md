# Beispielrechnung: Verbrennung von Methanol

Wir berechnen die Reaktionsenthalpie für die vollständige Verbrennung von Methanol.

## Die Reaktionsgleichung

$$\text{CH}_3\text{OH}(g) + \frac{3}{2}\text{O}_2(g) \rightarrow \text{CO}_2(g) + 2\text{H}_2\text{O}(g)$$

## Bildungsenthalpien (aus Tabelle)

| Stoff                     | $\Delta_f H°$ (kJ/mol) |
| ------------------------- | ---------------------- |
| $\text{CH}_3\text{OH}(g)$ | -201,5                 |
| $\text{O}_2(g)$           | 0 (Element!)           |
| $\text{CO}_2(g)$          | -393,5                 |
| $\text{H}_2\text{O}(g)$   | -241,8                 |

## Berechnung

### Schritt 1: Produkte (mit Koeffizienten)

$$\sum \Delta_f H°(\text{Produkte}) = 1 \cdot (-393,5) + 2 \cdot (-241,8)$$
$$= -393,5 - 483,6 = -877,1 \text{ kJ}$$

### Schritt 2: Edukte (mit Koeffizienten)

$$\sum \Delta_f H°(\text{Edukte}) = 1 \cdot (-201,5) + 1,5 \cdot 0$$
$$= -201,5 + 0 = -201,5 \text{ kJ}$$

### Schritt 3: Differenz berechnen

$$\Delta_r H° = \text{Produkte} - \text{Edukte}$$
$$\Delta_r H° = (-877,1) - (-201,5)$$
$$\boxed{\Delta_r H° = -675,6 \text{ kJ/mol}}$$

## Interpretation

- Das **negative Vorzeichen** zeigt: Die Reaktion ist **exotherm**
- Pro Mol verbranntem Methanol werden **675,6 kJ Energie** freigesetzt
- Das entspricht etwa dem Heizwert, der in Brennstofftabellen steht
