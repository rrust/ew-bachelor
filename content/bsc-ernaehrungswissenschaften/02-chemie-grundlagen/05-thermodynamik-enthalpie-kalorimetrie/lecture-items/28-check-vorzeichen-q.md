topic: 'Wärmekapazität und spezifische Wärme'
question: 'Was bedeutet ein negatives Vorzeichen bei $q$ ($q < 0$)?'
options:
  - 'Das System hat Wärme aufgenommen'
  - 'Die Temperatur ist gestiegen'
  - 'Das System hat Wärme abgegeben'
  - 'Die Wärmekapazität ist negativ'
correctAnswer: 'Das System hat Wärme abgegeben'
hint: 'Das Vorzeichen zeigt die Richtung des Energieflusses an.'
explanation: 'Vorzeichenkonvention: $q < 0$ bedeutet, dass Energie aus dem System heraus fließt - das System gibt Wärme ab und kühlt sich ab. Dies entspricht einer negativen Temperaturänderung ($\Delta T < 0$).'
