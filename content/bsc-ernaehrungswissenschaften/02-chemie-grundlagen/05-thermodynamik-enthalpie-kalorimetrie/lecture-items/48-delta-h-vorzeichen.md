# Das Vorzeichen von ΔH

Die Enthalpieänderung $\Delta H$ beschreibt die "Wärmeenergie", die ein System bei einer Reaktion austauscht.

## Die Grundformel

$$\Delta H = H_{\text{Produkte}} - H_{\text{Reaktanten}}$$

Oder in Worten: **Endzustand minus Anfangszustand**.

## Die Vorzeichen

| Situation                                     | Vorzeichen     | Bedeutung                                |
| --------------------------------------------- | -------------- | ---------------------------------------- |
| $H_{\text{Produkte}} < H_{\text{Reaktanten}}$ | $\Delta H < 0$ | **Exotherm** - Energie wird freigesetzt  |
| $H_{\text{Produkte}} > H_{\text{Reaktanten}}$ | $\Delta H > 0$ | **Endotherm** - Energie wird aufgenommen |

## Beispiele

### Exotherme Reaktion: Bildung von Wasser

$$\text{H}_2(g) + \frac{1}{2}\text{O}_2(g) \rightarrow \text{H}_2\text{O}(g)$$

$\Delta H = -241,8 \text{ kJ/mol}$

Die Produkte (Wasser) haben eine niedrigere Enthalpie als die Edukte. Die Differenz von 241,8 kJ wird pro Mol als Wärme freigesetzt.

### Endotherme Reaktion: Schmelzen von Eis

$$\text{H}_2\text{O}(s) \rightarrow \text{H}_2\text{O}(l)$$

$\Delta H = +6,01 \text{ kJ/mol}$

Das flüssige Wasser hat eine höhere Enthalpie als Eis. Die benötigte Energie wird der Umgebung entzogen.

## Merkhilfe

```text
ΔH negativ (-) → Energie geht raus → EXOtherm → Gefäß warm
ΔH positiv (+) → Energie geht rein → ENDOtherm → Gefäß kalt
```
