# Beispielrechnung: Abkühlung von Aluminium

Anhand eines konkreten Beispiels wenden wir die Formel $q = m \cdot c \cdot \Delta T$ an.

## Aufgabe

Wir kühlen 25,0 g Aluminium von 310 °C auf 37 °C ab. Wie viel Energie gibt das Aluminium ab?

## Gegeben

| Variable           | Wert          |
| ------------------ | ------------- |
| $m$                | 25,0 g        |
| $c_{\text{Al}}$    | 0,897 J/(g·K) |
| $T_{\text{start}}$ | 310 °C        |
| $T_{\text{end}}$   | 37 °C         |

## Rechnung

**Schritt 1:** Temperaturänderung berechnen
$$\Delta T = T_{\text{end}} - T_{\text{start}} = 37 - 310 = -273 \text{ K}$$

**Schritt 2:** Formel anwenden
$$q = m \cdot c \cdot \Delta T$$
$$q = 25,0 \text{ g} \cdot 0,897 \frac{\text{J}}{\text{g} \cdot \text{K}} \cdot (-273 \text{ K})$$

**Schritt 3:** Ausrechnen
$$q = -6120 \text{ J} = -6,12 \text{ kJ}$$

## Interpretation des Ergebnisses

Das **negative Vorzeichen** hat eine wichtige Bedeutung:

> **$q < 0$**: Das System (Aluminium) hat Wärme an die Umgebung **abgegeben**.

Das Aluminium hat 6,12 kJ Energie "verloren" - diese Energie wurde an die Umgebung übertragen.

## Merkhilfe

```text
q > 0 → Stoff nimmt Energie auf → wird wärmer
q < 0 → Stoff gibt Energie ab → wird kälter
```
