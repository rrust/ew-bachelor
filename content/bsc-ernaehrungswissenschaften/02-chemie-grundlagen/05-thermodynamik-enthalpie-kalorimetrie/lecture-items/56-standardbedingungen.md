# Standardbedingungen

Um thermodynamische Werte vergleichbar zu machen, definiert man **Standardbedingungen**.

## Definition der Standardbedingungen

| Parameter         | Standardwert      | Symbol           |
| ----------------- | ----------------- | ---------------- |
| **Druck**         | 1 atm = 101,3 kPa | ° (Hochstellung) |
| **Konzentration** | 1 mol/L           |                  |
| **Temperatur**    | 25 °C = 298 K     |                  |

## Das Standardsymbol

Standardwerte werden mit einem hochgestellten Kreis gekennzeichnet:

- $\Delta H°$ = Standardenthalpieänderung
- $\Delta_f H°$ = Standard-Bildungsenthalpie
- $\Delta_r H°$ = Standard-Reaktionsenthalpie

## Wichtige Hinweise

### Zur Temperatur

Die Standardtemperatur von 25°C ist **keine physikalische Notwendigkeit**, sondern eine praktische Konvention. Viele Tabellen geben Werte für 25°C an, weil dies nahe der Raumtemperatur liegt.

### Zum Aggregatzustand

Bei Standardbedingungen hat jeder Stoff seinen **charakteristischen Aggregatzustand**:

| Stoff       | Standardzustand bei 25°C          |
| ----------- | --------------------------------- |
| Wasser      | Flüssig ($\text{H}_2\text{O}(l)$) |
| Sauerstoff  | Gas ($\text{O}_2(g)$)             |
| Kohlenstoff | Fest, Graphit ($\text{C}(s)$)     |
| Eisen       | Fest ($\text{Fe}(s)$)             |
| Quecksilber | Flüssig ($\text{Hg}(l)$)          |

> **Merke:** Beim Berechnen von Reaktionsenthalpien muss der Aggregatzustand beachtet werden - er beeinflusst die $\Delta H$-Werte!
