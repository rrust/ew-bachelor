## Standardbedingungen in der Thermochemie

Um Enthalpiewerte vergleichbar zu machen, definiert man **Standardbedingungen**.

### Definition der Standardbedingungen

| Parameter         | Standardwert      | Symbol |
| ----------------- | ----------------- | ------ |
| **Druck**         | 1 atm (101,3 kPa) | °      |
| **Konzentration** | 1 mol/L           |        |
| **Temperatur**    | 25 °C (298 K)     |        |

Das hochgestellte ° (Kreis) zeigt an, dass es sich um Standardbedingungen handelt:
- $\Delta H°$ = Standard-Reaktionsenthalpie
- $\Delta_f H°$ = Standard-Bildungsenthalpie

### Standardzustand der Elemente

Jedes Element hat bei Standardbedingungen einen definierten **Referenzzustand**:

| Element     | Standardzustand            |
| ----------- | -------------------------- |
| Sauerstoff  | $O_2(g)$ – gasförmig       |
| Wasserstoff | $H_2(g)$ – gasförmig       |
| Stickstoff  | $N_2(g)$ – gasförmig       |
| Kohlenstoff | $C(s)$, Graphit – fest     |
| Schwefel    | $S_8(s)$ – fest, rhombisch |
| Phosphor    | $P_4(s)$ – fest, weiß      |
| Brom        | $Br_2(l)$ – flüssig        |
| Quecksilber | $Hg(l)$ – flüssig          |

### Warum ist das wichtig?

- Ermöglicht Vergleich von Literaturwerten
- Definierter Nullpunkt für Berechnungen
- Alle tabellierten Werte beziehen sich auf Standardbedingungen

> ⚠️ **Achtung:** Die Standardtemperatur 25 °C ist **nicht** "Normalbedingungen" (0 °C, 1 atm) aus der Physik!
