# Exotherme und Endotherme Prozesse

Je nach Richtung des Energieflusses unterscheiden wir zwei Typen von Prozessen.

## Exotherme Prozesse

Bei **exothermen** Prozessen fließt Energie vom SYSTEM zur UMGEBUNG.

| Merkmal             | Beobachtung                    |
| ------------------- | ------------------------------ |
| Energiefluss        | System → Umgebung              |
| Systemtemperatur    | Sinkt (Energie wird abgegeben) |
| Umgebungstemperatur | Steigt                         |
| Vorzeichen          | $\Delta H < 0$ (negativ)       |

**Beispiele:**
- Verbrennung von Brennstoffen
- Neutralisationsreaktionen
- Kristallisation aus übersättigter Lösung
- Kondensation von Wasserdampf

## Endotherme Prozesse

Bei **endothermen** Prozessen fließt Energie von der UMGEBUNG zum SYSTEM.

| Merkmal             | Beobachtung                       |
| ------------------- | --------------------------------- |
| Energiefluss        | Umgebung → System                 |
| Systemtemperatur    | Steigt (Energie wird aufgenommen) |
| Umgebungstemperatur | Sinkt                             |
| Vorzeichen          | $\Delta H > 0$ (positiv)          |

**Beispiele:**
- Verdampfen von Wasser
- Schmelzen von Eis
- Photosynthese
- Lösen von Ammoniumnitrat in Wasser (Kältepackung)

## Praktische Faustregel

> **Faustregel:** Wenn sich ein Reaktionsgefäß während der Reaktion **warm** anfühlt, ist der Prozess **exotherm**. Fühlt es sich **kalt** an, ist der Prozess **endotherm**.

Achtung: Wir fühlen immer die Temperatur der Umgebung (des Gefäßes), nicht des Systems!
