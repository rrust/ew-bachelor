# Standard-Bildungsenthalpie ($\Delta_f H°$)

Die Standard-Bildungsenthalpie ist ein zentraler Begriff in der Thermochemie.

## Definition

> Die **Standard-Bildungsenthalpie** $\Delta_f H°$ ist die Enthalpieänderung, wenn **1 Mol** einer Verbindung aus ihren **Elementen** im **Standardzustand** gebildet wird.

## Die goldene Regel

$$\boxed{\Delta_f H° = 0 \text{ für alle Elemente im Standardzustand}}$$

**Beispiele:**
- $\Delta_f H°(\text{O}_2, g) = 0$
- $\Delta_f H°(\text{C}, \text{Graphit}) = 0$
- $\Delta_f H°(\text{Fe}, s) = 0$
- $\Delta_f H°(\text{H}_2, g) = 0$

## Wichtige Bildungsenthalpien

| Verbindung                                       | $\Delta_f H°$ (kJ/mol) |
| ------------------------------------------------ | ---------------------- |
| $\text{H}_2\text{O}(l)$                          | -285,8                 |
| $\text{H}_2\text{O}(g)$                          | -241,8                 |
| $\text{CO}_2(g)$                                 | -393,5                 |
| $\text{CH}_4(g)$ (Methan)                        | -74,8                  |
| $\text{C}_2\text{H}_5\text{OH}(l)$ (Ethanol)     | -277,7                 |
| $\text{C}_6\text{H}_{12}\text{O}_6(s)$ (Glucose) | -1274                  |

## Positive vs. negative Werte

| $\Delta_f H°$ | Bedeutung                                      |
| ------------- | ---------------------------------------------- |
| Negativ       | Verbindung ist stabiler als die Elemente       |
| Positiv       | Verbindung ist weniger stabil als die Elemente |

> **Merke:** Die meisten stabilen Verbindungen haben **negative** Bildungsenthalpien - bei ihrer Bildung wird Energie frei.
