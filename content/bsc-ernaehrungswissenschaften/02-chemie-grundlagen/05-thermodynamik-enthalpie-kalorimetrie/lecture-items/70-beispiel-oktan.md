# Beispielrechnung: Verbrennung von Oktan

Eine vollständige Berechnung mit konkreten Zahlen.

## Gegeben

| Größe              | Wert     |
| ------------------ | -------- |
| Masse Oktan        | 1,00 g   |
| Masse Wasser       | 1200 g   |
| $C_{\text{Bombe}}$ | 837 J/K  |
| $T_{\text{start}}$ | 25,00 °C |
| $T_{\text{end}}$   | 33,20 °C |

## Berechnung

### Schritt 1: Temperaturänderung

$$\Delta T = 33,20 - 25,00 = 8,20 \text{ K}$$

### Schritt 2: Wärme des Wassers

$$q_{\text{Wasser}} = 1200 \text{ g} \cdot 4,184 \frac{\text{J}}{\text{g}\cdot\text{K}} \cdot 8,20 \text{ K}$$
$$q_{\text{Wasser}} = 41.170 \text{ J}$$

### Schritt 3: Wärme der Bombe

$$q_{\text{Bombe}} = 837 \frac{\text{J}}{\text{K}} \cdot 8,20 \text{ K} = 6.860 \text{ J}$$

### Schritt 4: Gesamtwärme

$$q_{\text{Gesamt}} = 41.170 + 6.860 = 48.030 \text{ J} = 48,0 \text{ kJ}$$

### Schritt 5: Reaktionsenergie

Da die Energie freigesetzt wurde (exotherm):
$$\Delta U = -48,0 \text{ kJ pro Gramm Oktan}$$

## Molare Verbrennungsenergie

Mit $M(\text{C}_8\text{H}_{18}) = 114 \text{ g/mol}$:

$$\Delta_c U_m = -48,0 \frac{\text{kJ}}{\text{g}} \cdot 114 \frac{\text{g}}{\text{mol}} = -5472 \text{ kJ/mol}$$

> **Interpretation:** Die Verbrennung von Oktan (ein Hauptbestandteil von Benzin) setzt enorme Energiemengen frei - daher ist es ein so effektiver Treibstoff!
