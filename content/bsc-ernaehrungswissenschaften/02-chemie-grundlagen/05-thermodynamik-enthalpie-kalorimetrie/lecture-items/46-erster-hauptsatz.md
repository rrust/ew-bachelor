# Der Erste Hauptsatz der Thermodynamik

Der Erste Hauptsatz verknüpft die Änderung der inneren Energie mit Wärme und Arbeit.

## Die fundamentale Gleichung

$$\boxed{\Delta U = q + w}$$

| Variable   | Bedeutung                    | Einheit |
| ---------- | ---------------------------- | ------- |
| $\Delta U$ | Änderung der inneren Energie | J       |
| $q$        | Übertragene Wärme            | J       |
| $w$        | Verrichtete Arbeit           | J       |

## Vorzeichenkonventionen

Diese Konvention ist entscheidend für alle Berechnungen:

### Für Wärme $q$

| Richtung                      | Vorzeichen | Prozesstyp |
| ----------------------------- | ---------- | ---------- |
| Energie HINEIN ins System     | $+q$       | Endotherm  |
| Energie HINAUS aus dem System | $-q$       | Exotherm   |

### Für Arbeit $w$

| Richtung                             | Vorzeichen | Prozesstyp  |
| ------------------------------------ | ---------- | ----------- |
| Arbeit am System verrichtet (HINEIN) | $+w$       | Kompression |
| Arbeit vom System geleistet (HINAUS) | $-w$       | Expansion   |

## In Worten

> **Der Erste Hauptsatz besagt:** Die Änderung der inneren Energie eines Systems entspricht der Summe aus zugeführter Wärme und verrichteter Arbeit.

## Energieerhaltung

Da Energie weder erzeugt noch vernichtet werden kann:

$$\Delta U_{\text{System}} + \Delta U_{\text{Umgebung}} = 0$$

Alles, was das System verliert, nimmt die Umgebung auf - und umgekehrt.
