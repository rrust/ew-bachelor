type: 'multiple-choice-multiple'
question: |
  Eisen (Fe) in Lebensmitteln liegt als Fe²⁺ (Häm-Eisen) oder Fe³⁺ (Nicht-Häm-Eisen) vor.
  Vitamin C (Ascorbinsäure) verbessert die Eisenresorption durch Reduktion von Fe³⁺ zu Fe²⁺.
  
  Welche chemischen Zusammenhänge sind korrekt?
options:
  - 'Bei der Reduktion von Fe³⁺ zu Fe²⁺ nimmt Eisen ein Elektron auf'
  - 'Vitamin C wirkt als Reduktionsmittel und wird dabei selbst oxidiert'
  - 'Die Oxidationszahl von Eisen ändert sich von +3 auf +2'
  - 'Häm-Eisen aus tierischen Quellen ist ein Komplex mit Fe im Zentrum'
  - 'Nicht-Häm-Eisen ist schwerer resorbierbar, weil Fe³⁺ in basischem Milieu unlöslich wird'
correctAnswers:
  - 'Bei der Reduktion von Fe³⁺ zu Fe²⁺ nimmt Eisen ein Elektron auf'
  - 'Vitamin C wirkt als Reduktionsmittel und wird dabei selbst oxidiert'
  - 'Die Oxidationszahl von Eisen ändert sich von +3 auf +2'
  - 'Häm-Eisen aus tierischen Quellen ist ein Komplex mit Fe im Zentrum'
  - 'Nicht-Häm-Eisen ist schwerer resorbierbar, weil Fe³⁺ in basischem Milieu unlöslich wird'
explanation: |
  **Elektronenaufnahme:** Reduktion = Elektronenaufnahme. Fe³⁺ + e⁻ → Fe²⁺ ✓
  
  **Vitamin C als Reduktionsmittel:** Das Reduktionsmittel gibt Elektronen ab und 
  wird dabei selbst oxidiert (Dehydroascorbinsäure entsteht). ✓
  
  **Oxidationszahländerung:** +3 → +2 ist eine Abnahme um 1, also Reduktion. ✓
  
  **Häm-Eisen:** Häm ist ein Porphyrin-Komplex mit Fe²⁺ im Zentrum (in Hämoglobin, Myoglobin). ✓
  
  **Fe³⁺-Löslichkeit:** Fe³⁺ bildet im basischen Milieu des Dünndarms schwerlösliches Fe(OH)₃,
  was die Resorption erschwert. Vitamin C hält das Eisen als Fe²⁺ löslich. ✓
