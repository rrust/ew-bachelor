# Was ist eine Lösung?

Eine **Lösung** ist ein **homogenes Gemisch** aus zwei oder mehr Substanzen, die in einer einzigen Phase vorliegen. Die Bestandteile sind auf molekularer Ebene so fein verteilt, dass sie optisch nicht mehr unterscheidbar sind.

## Die zwei Hauptkomponenten

| Komponente         | Bezeichnung | Beschreibung                                                |
| ------------------ | ----------- | ----------------------------------------------------------- |
| **Lösungsmittel**  | Solvens     | Meist im Überschuss vorhanden, bestimmt den Aggregatzustand |
| **Gelöster Stoff** | Solute      | Die im Lösungsmittel verteilten Komponenten                 |

> **Beispiel:** Bei einer Salzlösung ist Wasser das Lösungsmittel (Solvens) und NaCl der gelöste Stoff (Solute).

## Wichtige Begriffe

- **Homogen:** Gleichartig zusammengesetzt, keine Phasengrenzen erkennbar
- **Phase:** Ein Bereich mit einheitlichen physikalischen und chemischen Eigenschaften

Im Gegensatz zu **heterogenen Gemischen** (wie Suspensionen oder Emulsionen) zeigen Lösungen keine sichtbare Trennung der Bestandteile.
