# Massenanteil (Gewichts-%)

Der **Massenanteil** gibt an, wie viel Prozent der Gesamtmasse einer Lösung vom gelösten Stoff stammt.

## Definition

$$\text{Massenanteil} = \frac{m_{\text{gelöster Stoff}}}{m_{\text{Lösung}}} \times 100\%$$

Wobei $m_{\text{Lösung}} = m_{\text{gelöster Stoff}} + m_{\text{Lösungsmittel}}$

## Eigenschaften

| Eigenschaft          | Wert                      |
| -------------------- | ------------------------- |
| Einheit              | % (Prozent)               |
| Andere Bezeichnungen | Gewichts-%, Gew.-%, w/w-% |

## Alltagsbeispiele

| Lösung                        | Massenanteil       |
| ----------------------------- | ------------------ |
| Physiologische Kochsalzlösung | 0,9% NaCl          |
| Essig                         | ca. 5% Essigsäure  |
| Konzentrierte Schwefelsäure   | 98% H₂SO₄          |
| Haushaltszucker-Sirup         | ca. 66% Saccharose |

> **Beachte:** Der Massenanteil sagt nichts über die Stoffmenge aus. 10% Glucose (M = 180 g/mol) entspricht einer anderen Molzahl als 10% NaCl (M = 58 g/mol).
