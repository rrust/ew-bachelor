# Stoffmengenanteil (Mol-Bruch)

Der **Stoffmengenanteil** $X$ (auch Mol-Bruch genannt) gibt das Verhältnis der Stoffmenge einer Komponente zur Gesamtstoffmenge an.

## Definition

$$X_A = \frac{n_A}{n_A + n_B + n_C + \dots} = \frac{n_A}{n_{\text{gesamt}}}$$

## Eigenschaften

| Eigenschaft   | Wert                                   |
| ------------- | -------------------------------------- |
| Einheit       | **dimensionslos** (mol/mol kürzt sich) |
| Wertebereich  | 0 bis 1                                |
| Summe aller X | Immer 1                                |

## Beispiel

In einer Mischung mit 2 mol Ethanol und 8 mol Wasser:

$$X_{\text{Ethanol}} = \frac{2}{2 + 8} = \frac{2}{10} = 0{,}2$$

$$X_{\text{Wasser}} = \frac{8}{10} = 0{,}8$$

> **Prüfung:** $X_{\text{Ethanol}} + X_{\text{Wasser}} = 0{,}2 + 0{,}8 = 1$ ✓
